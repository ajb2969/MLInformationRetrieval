Path-connectedness, completeness and geodesic completeness

It can be shown that a finite-dimensional path-connected Riemannian manifold is a complete metric space (with respect to the Riemannian distance) if and only if it is geodesically complete. This is the Hopf‚ÄìRinow theorem. This theorem does not hold for infinite-dimensional manifolds. The example of a non-complete manifold (the punctured plane) given above fails to be geodesically complete because, although it is path-connected, it is not a complete metric space: any sequence in the plane converging to the origin is a non-converging Cauchy sequence in the punctured plane.
