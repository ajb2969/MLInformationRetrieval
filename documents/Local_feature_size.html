Local feature size

Local feature size refers to several related concepts in computer graphics and computational geometry for measuring the size of a geometric object near a particular point.

  • Given a smooth manifold M M M , the local feature size at any point x ∈ M x M x\in M is the distance between x x x and the medial axis of M M M .1
  • Given a planar straight-line graph, the local feature size at any point x x x is the radius of the smallest closed ball centered at x x x which intersects any two disjoint features (vertices or edges) of the graph.2
