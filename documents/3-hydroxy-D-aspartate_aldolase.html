3-hydroxy-D-aspartate aldolase

3-hydroxy-D-aspartate aldolase (, D-3-hydroxyaspartate aldolase) is an enzyme with system name 3-hydroxy-D-aspartate glyoxylate-lyase (glycine-forming).1 This enzyme catalyses the following chemical reaction

(1) threo-3-hydroxy-D-aspartate ⇌ normal-⇌ \rightleftharpoons glycine + glyoxylate
(2) D-erythro-3-hydroxyaspartate ⇌ normal-⇌ \rightleftharpoons glycine + glyoxylate

This enzyme is a pyridoxal-phosphate protein.
