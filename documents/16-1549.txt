The tool uses JavaScript by default. Check your various JavaScript configurations at . If an invalid entry is added, it will break all uses of JS but since JS is enabled it will not fallback to MediaWiki:Edittools . Personal JS pages will now show an error on save if there is an invalid entry, so an attempt to edit and save will point out issues.   Technical  If JavaScript is enabled, then CharInsert is added as a Gadget defined at MediaWiki:Gadget-charinsert.js which uses JavaScript to load CharInsert from MediaWiki:Gadget-charinsert-core.js . The Gadget is enabled by default but can be disabled through .  If JavaScript is disabled, then the CharInsert software extension is called, which uses the symbols defined at MediaWiki:Edittools . Not all of the symbol sets are included in MediaWiki:Edittools. This mode is also used if the bits server fails to yield all the needed Javascript and CSS.
