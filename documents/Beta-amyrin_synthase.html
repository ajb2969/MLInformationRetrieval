Beta-amyrin synthase

Beta-amyrin synthase (, 2,3-oxidosqualene beta-amyrin cyclase, AsbAS1, BPY, EtAS, GgbAS1, LjAMY1, MtAMY1, PNY, BgbAS) is an enzyme with system name (3S)-2,3-epoxy-2,3-dihydrosqualene mutase (cyclizing, beta-amyrin-forming).1234567891011 This enzyme catalyses the following chemical reaction

(3S)-2,3-epoxy-2,3-dihydrosqualene ⇌ normal-⇌ \rightleftharpoons beta-amyrin

Some organism possess a monofunctional beta-amyrin synthase.
