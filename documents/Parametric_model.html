Parametric model

Parametric models are contrasted with the semi-parametric, semi-nonparametric, and non-parametric models, all of which consist of an infinite set of ‚Äúparameters‚Äù for description. The distinction between these four classes is as follows:

  • in a ‚Äúparametric‚Äù model all the parameters are in finite-dimensional parameter spaces;
  • a model is ‚Äúnon-parametric‚Äù if all the parameters are in infinite-dimensional parameter spaces;
  • a ‚Äúsemi-parametric‚Äù model contains finite-dimensional parameters of interest and infinite-dimensional nuisance parameters;
  • a ‚Äúsemi-nonparametric‚Äù model has both finite-dimensional and infinite-dimensional unknown parameters of interest.
