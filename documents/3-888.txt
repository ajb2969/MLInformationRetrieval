   Mertens function      Mertens function   In number theory , the Mertens function is defined for all positive integers  n as       M   (  n  )    =    ∑   k  =  1   n    μ   (  k  )           M  n     superscript   subscript     k  1    n     μ  k      M(n)=\sum_{k=1}^{n}\mu(k)     where μ(k) is the Möbius function . The function is named in honour of Franz Mertens .  Less formally, M ( n ) is the count of square-free integers up to n that have an even number of prime factors, minus the count of those that have an odd number. Because the Möbius function only takes the values −1, 0, and +1, the Mertens function moves slowly and there is no n such that | M ( n )| > n . The Mertens conjecture went further, stating that there would be no n where the absolute value of the Mertens function exceeds the square root of n . The Mertens conjecture was proven false in 1985 by Andrew Odlyzko and Herman te Riele . However, the Riemann hypothesis is equivalent to a weaker conjecture on the growth of M ( n ), namely M ( n ) = O ( n 1/2 + ε ). Since high values for M ( n ) grow at least as fast as the square root of n , this puts a rather tight bound on its rate of growth. Here, O refers to Big O notation . Calculation  Neither of the methods mentioned previously leads to practical algorithms to calculate the Mertens function.
