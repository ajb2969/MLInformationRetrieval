   Matrix splitting      Matrix splitting   In the mathematical discipline of numerical linear algebra , a matrix splitting is an expression which represents a given matrix as a sum or difference of matrices. Many iterative methods (e.g., for systems of differential equations ) depend upon the direct solution of matrix equations involving matrices more general than tridiagonal matrices . These matrix equations can often be solved directly and efficiently when written as a matrix splitting. The technique was devised by Richard S. Varga in 1960. 1  Regular splittings  We seek to solve the matrix equation       𝔸  𝕩   =   𝕜  ,   (  1  )          𝔸  𝕩    𝕜  1     \mathbb{A}\mathbb{x}=\mathbb{k},\quad(1)     where A is a given n × n  non-singular matrix, and k is a given column vector with n components. We split the matrix A into      𝔸  =    𝔹  -  ℂ   ,   (  2  )        𝔸     𝔹  ℂ   2     \mathbb{A}=\mathbb{B}-\mathbb{C},\quad(2)     where B and C are n × n matrices. If, for an arbitrary n × n matrix M , M has nonnegative entries, we write M ≥ 0 . If M has only positive entries, we write M > 0 . Similarly, if the matrix M 1 − M 2 has nonnegative entries, we write M 1 ≥ M 2 .  Definition: A = B − C is a regular splitting of A if and only if B −1 ≥ 0 and C ≥ 0 .
