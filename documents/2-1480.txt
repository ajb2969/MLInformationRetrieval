Basic properties   The complement of a G Î´ set is an F Ïƒ set.    The intersection of countably many G Î´ sets is a G Î´ set, and the union of finitely many G Î´ sets is a G Î´ set; a countable union of G Î´ sets is called a G Î´Ïƒ set.    In metrizable spaces, every closed set is a G Î´ set and, dually, every open set is an F Ïƒ set.    A subspace  A of a completely metrizable space X is itself completely metrizable if and only if A is a G Î´ set in X .    A set that contains the intersection of a countable collection of dense open sets is called comeagre or residual. These sets are used to define generic properties of topological spaces of functions. G Î´ space  A G Î´ space is a topological space in which every closed set is a G Î´ set . A normal space which is also a G Î´ space is perfectly normal .
