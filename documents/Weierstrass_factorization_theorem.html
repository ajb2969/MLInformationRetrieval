Weierstrass factorization theorem

In mathematics, and particularly in the field of complex analysis, the Weierstrass factorization theorem asserts that entire functions can be represented by a product involving their zeroes. In addition, every sequence tending to infinity has an associated entire function with zeroes at precisely the points of that sequence. The theorem is named after Karl Weierstrass.

A second form of the theorem extends to meromorphic functions and allows one to consider a given meromorphic function as a product of three factors: terms depending on the function's poles and zeroes, and an associated non-zero holomorphic function.

The two forms of the Weierstrass factorization theorem can be thought of as extensions of the above to entire functions. The necessity of extra machinery is demonstrated when one considers the product ∏ n ( z - c n ) subscript product n z subscript c n \,\prod_{n}(z-c_{n}) if the sequence { c n } subscript c n \{c_{n}\} is not finite. It can never define an entire function, because the infinite product does not converge. Thus one cannot, in general, define an entire function from a sequence of prescribed zeroes or represent an entire function by its zeroes using the expressions yielded by the fundamental theorem of algebra.

A necessary condition for convergence of the infinite product in question is that each factor ( z - c n ) z subscript c n (z-c_{n}) must approach 1 as n → ∞ normal-→ n n\to\infty . So it stands to reason that one should seek a function that could be 0 at a prescribed point, yet remain near 1 when not at that point and furthermore introduce no more zeroes than those prescribed. Weierstrass' elementary factors have these properties and serve the same purpose as the factors ( z - c n ) z subscript c n (z-c_{n}) above.
