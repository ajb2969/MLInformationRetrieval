Farnesylcysteine lyase

Farnesylcysteine lyase (, FC lyase, FCLY) is an enzyme with system name S-(2E,6E)-farnesyl-L-cysteine oxidase.12 This enzyme catalyses the following chemical reaction

S-(2E,6E)-farnesyl-L-cysteine + O2 + H2O ⇌ normal-⇌ \rightleftharpoons (2E,6E)-farnesal + L-cysteine + H2O2

Farnesylcysteine lyase is a flavoprotein (FAD).
