Verifiable random function

In cryptography, the concept of a verifiable random function was introduced by Micali, Rabin, and Vadhan.1 It is a pseudo-random function that provides publicly verifiable proofs of its outputs' correctness. Given an input value x, the owner of the secret key SK can compute the function value y = FSK(x) and the proof pSK(x).
