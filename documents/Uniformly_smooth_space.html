Enflo proved6 that the class of Banach spaces that admit an equivalent uniformly convex norm coincides with the class of super-reflexive Banach spaces, introduced by Robert C. James.7 As a space is super-reflexive if and only if its dual is super-reflexive, it follows that the class of Banach spaces that admit an equivalent uniformly convex norm coincides with the class of spaces that admit an equivalent uniformly smooth norm. The Pisier renorming theorem8 states that a super-reflexive space X admits an equivalent uniformly smooth norm for which the modulus of smoothness ρX satisfies, for some constant C and some

If a normed space admits two equivalent norms, one uniformly convex and one uniformly smooth, the Asplund averaging technique9 produces another equivalent norm that is both uniformly convex and uniformly smooth.
