Psi function

Psi function can refer, in mathematics, to

  • the ordinal collapsing function ψ ( α ) ψ α \psi(\alpha)
  • the Dedekind psi function ψ ( n ) ψ n \psi(n)
  • the Chebyshev function ψ ( x ) ψ x \psi(x)
  • the polygamma function ψ m ( z ) superscript ψ m z \psi^{m}(z) or its special cases
    • the digamma function ψ ( z ) ψ z \psi(z)
    • the trigamma function ψ 1 ( z ) superscript ψ 1 z \psi^{1}(z)

and in physics to

  • the quantum mechanical wave function.
