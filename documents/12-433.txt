If at least one of Laplace invariants is not equal to zero, i.e.         c  -   a  b   -   a  x    ≠   0  and/or      c  -   a  b   -   b  y    ≠  0    ,     formulae-sequence      c    a  b    subscript  a  x     0  and/or        c    a  b    subscript  b  y    0     c-ab-a_{x}\neq 0\quad\text{and/or}\quad c-ab-b_{y}\neq 0,     then this representation is a first step of the Laplace–Darboux transformations used for solving non-factorizable bivariate linear partial differential equations (LPDEs).  If both Laplace invariants are equal to zero, i.e.         c  -   a  b   -   a  x    =   0  and      c  -   a  b   -   b  y    =  0    ,     formulae-sequence      c    a  b    subscript  a  x     0  and        c    a  b    subscript  b  y    0     c-ab-a_{x}=0\quad\text{and}\quad c-ab-b_{y}=0,     then the differential operator A is factorizable and corresponding linear partial differential equation of second order is solvable.  Laplace invariants have been introduced for a bivariate linear partial differential operator (LPDO) of order 2 and of hyperbolic type. They are a particular case of generalized invariants which can be constructed for a bivariate LPDO of arbitrary order and arbitrary type; see Invariant factorization of LPDOs .
