Related distributions

  • If X X X is a random variable with the non-central chi distribution, the random variable X 2 superscript X 2 X^{2} will have the noncentral chi-squared distribution. Other related distributions may be seen there.
  • If X X X is chi distributed X ∼ χ k similar-to X subscript χ k X\sim\chi_{k} then X X X is also non-central chi distributed X ∼ N C χ k ( 0 ) similar-to X N C subscript χ k 0 X\sim NC\chi_{k}(0) . In other words, the chi distribution is a special case of the non-central chi distribution (i.e., with a non-centrality parameter of zero).
  • A noncentral chi distribution with 2 degrees of freedom is equivalent to a Rice distribution with σ = 1 σ 1 \sigma=1 .
  • If X follows a noncentral chi distribution with 1 degree of freedom and noncentrality parameter λ, then σX follows a folded normal distribution whose parameters are equal to σλ and σ2 for any value of σ.

Applications

The Euclidean norm of a multivariate normally distributed random vector follows a noncentral chi distribution.
