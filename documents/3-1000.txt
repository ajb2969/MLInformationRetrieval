   Integral test for convergence      Integral test for convergence     In mathematics , the integral test for convergence is a method used to test infinite series of non-negative terms for convergence . It was developed by Colin Maclaurin and Augustin-Louis Cauchy and is sometimes known as the Maclaurin–Cauchy test.  Statement of the test  Consider an integer    N   N   N   and a non-negative function   f   f   f   defined on the unbounded interval     [  N  ,  ∞  )     N  normal-∞    [N,∞)   , on which it is monotone decreasing . Then the infinite series       ∑   n  =  N   ∞    f   (  n  )        superscript   subscript     n  N        f  n     \sum_{n=N}^{\infty}f(n)     converges to a real number if and only if the improper integral       ∫  N  ∞    f   (  x  )   d  x       superscript   subscript   N       f  x  d  x     \int_{N}^{\infty}f(x)\,dx     is finite. In other words, if the integral diverges, then the series diverges as well.
