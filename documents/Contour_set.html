Contour set

In mathematics, contour sets generalize and formalize the everyday notions of

  • everything superior to something
  • everything superior or equivalent to something
  • everything inferior to something
  • everything inferior or equivalent to something.

Economic

In economics, the set X X X could be interpreted as a set of goods and services or of possible outcomes, the relation ≻ succeeds \succ as strict preference, and the relationship ≽ succeeds-or-equals \succcurlyeq as weak preference. Then

  • the upper contour set, or better set,1 of x x x would be the set of all goods, services, or outcomes that were at least as desired as x x x ,
  • the strict upper contour set of x x x would be the set of all goods, services, or outcomes that were more desired than x x x ,
  • the lower contour set, or worse set,2 of x x x would be the set of all goods, services, or outcomes that were no more desired than x x x , and
  • the strict lower contour set of x x x would be the set of all goods, services, or outcomes that were less desired than x x x .
