   Ruelle zeta function      Ruelle zeta function   In mathematics , the Ruelle zeta function is a zeta function associated with a dynamical system .  Formal definition  Let f be a function defined on a manifold  M , such that the set of fixed points Fix( f  n ) is finite for all n > 1. Further let φ be a function on M with values in d × d complex matrices.
