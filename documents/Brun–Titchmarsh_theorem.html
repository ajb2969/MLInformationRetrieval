Brun–Titchmarsh theorem

In analytic number theory, the Brun–Titchmarsh theorem, named after Viggo Brun and Edward Charles Titchmarsh, is an upper bound on the distribution of prime numbers in arithmetic progression.

Statement

It states that, if π ( x ; q , a ) π x q a \pi(x;q,a) counts the number of primes p congruent to a modulo q with p ≤ x, then

π ( x ; q , a ) ≤ 2 x φ ( q ) log ( x / q ) π x q a 2 x φ q x q \pi(x;q,a)\leq{2x\over\varphi(q)\log(x/q)}

for all q 1+o(1).
