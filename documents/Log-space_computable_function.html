Log-space computable function

A log-space computable function is a function f : Σ ∗ → Σ ∗ normal-: f normal-→ superscript normal-Σ normal-∗ superscript normal-Σ normal-∗ f\colon\Sigma^{\ast}\rightarrow\Sigma^{\ast} that requires only O ( log n ) O n O(\log n) memory to be computed (this restriction does not apply to the size of the output). The computation is generally done by means of a log-space transducer.

Log-space reductions

The main use for log-space computable functions is in log-space reductions. This is a means of transforming an instance of one problem into an instance of another problem, using only logarithmic space.
