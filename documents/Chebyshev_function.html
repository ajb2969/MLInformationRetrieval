The Chebyshev functions, especially the second one ψ(x), are often used in proofs related to prime numbers, because it is typically simpler to work with them than with the prime-counting function, π(x) (See the exact formula, below.) Both Chebyshev functions are asymptotic to x, a statement equivalent to the prime number theorem.

Both functions are named in honour of Pafnuty Chebyshev.

Properties

A theorem due to Erhard Schmidt states that, for some explicit positive constant K, there are infinitely many natural numbers x such that

ψ ( x ) - x < - K x ψ x x K x \psi(x)-x<-K\sqrt{x}

and infinitely many natural numbers x such that

ψ ( x ) - x > K x . ψ x x K x \psi(x)-x>K\sqrt{x}.

In little-o notation, one may write the above as

ψ ( x ) - x ≠ o ( x ) . ψ x x o x \psi(x)-x\neq o\left(\sqrt{x}\right).
