Matrix product state

Matrix product state (MPS) is a pure quantum state of many particles, written in the following form:

where A i ( s ) superscript subscript A i s A_{i}^{(s)} are complex, square matrices of order χ χ \chi (this dimension is called local dimension). Indices s i subscript s i s_{i} go over states in the computational basis. For qubits, it is s i ∈ { 0 , 1 } subscript s i 0 1 s_{i}\in\{0,1\} . For qudits (d-level systems), it is s i ∈ { 0 , 1 , … , d - 1 } subscript s i 0 1 normal-… d 1 s_{i}\in\{0,1,\ldots,d-1\} .

It is particularly useful for dealing with ground states of one-dimensional quantum spin models (e.g. Heisenberg model (quantum)). The parameter χ χ \chi is related to entanglement between particles. In particular, if the state is a product state (i.e. not entangled at all), it can be described as a matrix product state with χ = 1 χ 1 \chi=1 .

For states that are translationally symmetric, we can choose:

In general, every state can be written in the MPS form (with χ χ \chi growing exponentially with the particle number N). However, MPS are practical when χ χ \chi is small – for example, does not depend on the particle number. Except for a small number of specific cases (some mentioned in the section Examples), such thing is not possible. Though, in many cases it serves as a good approximation.

MPS decomposition is not unique.
