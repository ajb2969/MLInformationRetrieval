   Involutory matrix      Involutory matrix   In mathematics , an involutory matrix is a matrix that is its own inverse. That is, multiplication by matrix A is an involution if and only if A 2 = I . Involutory matrices are all square roots of the identity matrix . Clearly, any block-diagonal matrices constructed from involutory matrices will also be involutory, as a consequence of the linear independence of the blocks.  Symmetry  An involutory matrix which is also symmetric is an orthogonal matrix , and thus represents an isometry (a linear transformation which preserves Euclidean distance ). Conversely every orthogonal involutory matrix is symmetric. 3 As a special case of this, every reflection matrix is a involutory.
