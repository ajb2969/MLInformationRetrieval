   Dependability state model      Dependability state model   A dependability state diagram is a method for modelling a system as a Markov chain . It is used in reliability engineering for availability and reliability analysis. 1   It consists of creating a finite state machine which represent the different states a system may be in. Transitions between states happen as a result of events from underlying Poisson processes with different intensities. Availability  The asymptotic availability , i.e. availability over a long period, of the system is equal to the probability that the model is in state 1 or state 2.  This is calculated by making a set of linear equations of the state transition and solving the linear system.  The matrix is constructed with a row for each state.
