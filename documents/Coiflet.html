Coiflet

Coiflets are discrete wavelets designed by Ingrid Daubechies, at the request of Ronald Coifman, to have scaling functions with vanishing moments. The wavelet is near symmetric, their wavelet functions have N / 3 N 3 N/3 vanishing moments and scaling functions N / 3 - 1 N 3 1 N/3-1 , and has been used in many applications using Calderón-Zygmund Operators.12

Coiflet coefficients

Both the scaling function (low-pass filter) and the wavelet function (High-Pass Filter) must be normalised by a factor 1 / 2 1 2 1/\sqrt{2} . The wavelet coefficients are derived by reversing the order of the scaling function coefficients and then reversing the sign of every second one (i.e. C6 wavelet = {−0.022140543057, 0.102859456942, 0.544281086116, −1.205718913884, 0.477859456942, 0.102859456942}).

Matlab function

F = coifwavf(W) returns the scaling filter associated with the Coiflet wavelet specified by the string W where W = 'coifN'. Possible values for N are 1, 2, 3, 4, or 5. 4
