   Kuratowski closure axioms      Kuratowski closure axioms   In topology and related branches of mathematics , the Kuratowski closure axioms are a set of axioms that can be used to define a topological structure on a set . They are equivalent to the more commonly used open set definition. They were first introduced by Kazimierz Kuratowski . 1  A similar set of axioms can be used to define a topological structure using only the dual notion of interior operator .
