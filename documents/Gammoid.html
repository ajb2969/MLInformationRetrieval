Gammoid

In matroid theory, a field within mathematics, a gammoid is a certain kind of matroid, describing sets of vertices that can be reached by vertex-disjoint paths in a directed graph.

The concept of a gammoid was introduced and shown to be a matroid by , based on considerations related to Menger's theorem characterizing the obstacles to the existence of systems of disjoint paths.1 Gammoids were studied in more detail and given their name by .2

Definition

Let G G G be a directed graph, S S S be a set of starting vertices, and T T T be a set of destination vertices (not necessarily disjoint from S S S ). The gammoid Γ normal-Γ \Gamma derived from this data has T T T as its set of elements. A subset I I I of T T T is independent in Γ normal-Γ \Gamma if there exists a set of vertex-disjoint paths whose starting points all belong to S S S and whose ending points are exactly I I I .3

A strict gammoid is a gammoid in which the set T T T of destination vertices consists of every vertex in G G G . Thus, a gammoid is a restriction of a strict gammoid, to a subset of its elements.45

Example

Consider the uniform matroid U n r fragments U superscript subscript absent n r U{}^{r}_{n} on a set of n n n elements, in which every set of r r r or fewer elements is independent. One way to represent this matroid as a gammoid would be to form a complete bipartite graph K r , n subscript K r n K_{r,n} with a set S S S of r r r vertices on one side of the bipartition, with a set T T T of n n n vertices on the other side of the bipartition, and with every edge directed from S S S to T T T . In this graph, a subset of T T T is the set of endpoints of a set of disjoint paths if and only if it has r r r or fewer vertices, for otherwise there aren't enough vertices in S S S to start the paths. The special structure of this graph shows that the uniform matroid is a transversal matroid as well as being a gammoid.6

Alternatively, the same uniform matroid U n r fragments U superscript subscript absent n r U{}^{r}_{n} may be represented as a gammoid on a smaller graph, with only n n n vertices, by choosing a subset S S S of r r r vertices and connecting each of the chosen vertices to every other vertex in the graph. Again, a subset of the vertices of the graph can be endpoints of disjoint paths if and only if it has r r r or fewer vertices, because otherwise there are not enough vertices that can be starts of paths. In this graph, every vertex corresponds to an element of the matroid, showing that the uniform matroid is a strict gammoid.7

Menger's theorem and gammoid rank

The rank of a set X ⊂ T X T X\subset T in a gammoid defined from a graph G G G and vertex subsets S S S and T T T is, by definition, the maximum number of vertex-disjoint paths from S S S to X X X . By Menger's theorem, it also equals the minimum cardinality of a set Y Y Y that intersects every path from S S S to X X X .8

Relation to transversal matroids

A transversal matroid is defined from a family of sets: its elements are the elements of the sets, and a set X X X of these elements is independent whenever there exists a one-to-one matching of the elements of X X X to disjoint sets containing them, called a system of distinct representatives. Representability

It is not true that every gammoid is regular, i.e., representable over every field. In particular, the uniform matroid U 4 2 fragments U superscript subscript absent 4 2 U{}^{2}_{4} is not a binary matroid, and more generally the n n n -point line U n 2 fragments U superscript subscript absent n 2 U{}^{2}_{n} can only be represented over fields with n - 1 n 1 n-1 or more elements. However, every gammoid may be represented over almost every finite field.
