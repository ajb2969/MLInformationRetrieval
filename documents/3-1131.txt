   Module homomorphism      Module homomorphism   In algebra, a module homomorphism is a function between modules that preserves module structures. Explicitly, if M and N are left modules over a ring R , then a function    f  :   M  →  N      normal-:  f   normal-→  M  N     f:M\to N   is called a module homomorphism or a R -linear map if for any x , y in M and r in R ,        f   (   x  +  y   )    =    f   (  x  )    +   f   (  y  )      ,        f    x  y        f  x     f  y      f(x+y)=f(x)+f(y),           f   (   r  x   )    =   r  f   (  x  )     .        f    r  x      r  f  x     f(rx)=rf(x).   If M , N are right module, then the second condition is replaced with          f   (   x  r   )    =   f   (  x  )   r    .        f    x  r      f  x  r     f(xr)=f(x)r.        The pre-image of the zero element under f is called the kernel of f . The set of all module homomorphisms from M to N is denoted by Hom R ( M , N ). It is an abelian group but is not necessarily a module unless R is commutative.  The isomorphism theorems hold for module homomorphisms. Exact sequences  A short sequence of modules over a commutative ring      0  →   A   →  𝑓   B   →  𝑔   C   →  0       normal-→  0    A   f  normal-→   B   g  normal-→   C     normal-→    0     0\to A\overset{f}{\to}B\overset{g}{\to}C\to 0   consists of modules A , B , C and homomorphisms f , g . It is exact if f is injective, the kernel of g is the image of f and g is surjective. A longer exact sequence is defined in the similar way. A sequence of modules is exact if and only if it is exact as a sequence of abelian groups.
