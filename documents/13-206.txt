Talalaev solved the long-standing problem of the explicit solution for the full set of the quantum commuting conservation laws for the Gaudin model, discovering the following theorem.  Consider  $$\det\left(\frac{\partial}{\partial_z} - L(z) \right) =\sum_{i=0}^n H_i(z) \left(\frac{\partial}{\partial_z}\right)^i.$$  Then for all i,j,z,w  $$[ H_i(z), H_j(w) ]= 0, ~~~~~~~~$$ i.e. H i ( z ) are generating functions in z for the differential operators in x which all commute. So they provide quantum commuting conservation laws for the Gaudin model.
