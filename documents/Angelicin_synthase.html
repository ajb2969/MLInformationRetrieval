Angelicin synthase

Angelicin synthase (, CYP71AJ4 (gene)) is an enzyme with system name ''(+)-columbianetin,NADPH:oxygen oxidoreductase ''.1 This enzyme catalyses the following chemical reaction

(+)-columbianetin + NADPH + H+ + O2 ⇌ normal-⇌ \rightleftharpoons angelicin + NADP+ + acetone + 2 H2O
