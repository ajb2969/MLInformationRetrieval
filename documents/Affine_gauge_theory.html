Affine gauge theory

Affine gauge theory is classical gauge theory where gauge fields are affine connections on the tangent bundle over a smooth manifold X X X . For instance, these are gauge theory of dislocations in continuous media when X = ℝ 3 X superscript ℝ 3 X=\mathbb{R}^{3} , the generalization of metric-affine gravitation theory when X X X is a world manifold and, in particular, gauge theory of the fifth force.

Affine tangent bundle

Being a vector bundle, the tangent bundle T X T X TX of an n n n -dimensional manifold X X X admits a natural structure of an affine bundle A T X A T X ATX , called the affine tangent bundle, possessing bundle atlases with affine transition functions. It is associated to a principal bundle A F X A F X AFX of affine frames in tangent space over X X X , whose structure group is a general affine group G A ( n , ℝ ) G A n ℝ GA(n,\mathbb{R}) .

The tangent bundle T X T X TX is associated to a principal linear frame bundle F X F X FX , whose structure group is a general linear group G L ( n , ℝ ) G L n ℝ GL(n,\mathbb{R}) . This is a subgroup of G A ( n , ℝ ) G A n ℝ GA(n,\mathbb{R}) so that the latter is a semidirect product of G L ( n , ℝ ) G L n ℝ GL(n,\mathbb{R}) and a group T n superscript T n T^{n} of translations.

There is the canonical imbedding of F X F X FX to A F X A F X AFX onto a reduced principal subbundle which corresponds to the canonical structure of a vector bundle T X T X TX as the affine one.
