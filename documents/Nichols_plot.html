Nichols plot

The Nichols plot is a plot used in signal processing and control design, named after American engineer Nathaniel B. Nichols.123

In feedback control design, the plot is useful for assessing the stability and robustness of a linear system. This application of the Nichols plot is central to the Quantitative feedback theory (QFT) of Horowitz and Sidi, which is a well known method for robust control system design.

In most cases, arg ( G ( s ) ) G s \arg(G(s)) refers to the phase of the system's response. Although similar to a Nyquist plot, a Nichols plot is plotted in a Cartesian coordinate system while a Nyquist plot is plotted in a polar coordinate system.
