Uses

One of the main applications of this function is in the resolution of the equation z = ln(z), as the only solution is given by z = e−ω(π i).

y = ω(z) is the unique solution, when z ≠ x ± i π z plus-or-minus x i π z\neq x\pm i\pi for x ≤ −1, of the equation y + ln(y) = z. Except on those two rays, the Wright omega function is continuous, even analytic.
