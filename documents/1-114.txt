   Biconditional elimination      Biconditional elimination   Biconditional elimination is the name of two valid  rules of inference of propositional logic . It allows for one to infer a conditional from a biconditional . If    (   P  ↔  Q   )     normal-↔  P  Q    (P\leftrightarrow Q)   is true, then one may infer that    (   P  →  Q   )     normal-→  P  Q    (P\to Q)   is true, and also that    (   Q  →  P   )     normal-→  Q  P    (Q\to P)   is true. 1 For example, if it's true that I'm breathing if and only if I'm alive, then it's true that if I'm breathing, I'm alive; likewise, it's true that if I'm alive, I'm breathing.
