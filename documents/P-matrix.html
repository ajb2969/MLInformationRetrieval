Remarks

The class of nonsingular M-matrices is a subset of the class of P P P -matrices. More precisely, all matrices that are both P P P -matrices and Z-matrices are nonsingular M M M -matrices. The class of sufficient matrices is another generalization of P P P -matrices.1

If the Jacobian of a function is a P P P -matrix, then the function is injective on any rectangular region of ℝ n superscript ℝ n \mathbb{R}^{n} .

A related class of interest, particularly with reference to stability, is that of P ( - ) superscript P P^{(-)} -matrices, sometimes also referred to as N - P N P N-P -matrices. A matrix A A A is a P ( - ) superscript P P^{(-)} -matrix if and only if ( - A ) A (-A) is a P P P -matrix (similarly for P 0 subscript P 0 P_{0} -matrices). Since σ ( A ) = - σ ( - A ) σ A σ A \sigma(A)=-\sigma(-A) , the eigenvalues of these matrices are bounded away from the positive real axis.
