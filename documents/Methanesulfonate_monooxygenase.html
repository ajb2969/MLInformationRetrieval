Methanesulfonate monooxygenase

Methanesulfonate monooxygenase (, mesylate monooxygenase, mesylate,reduced-FMN:oxygen oxidoreductase, MsmABCD, methanesulfonic acid monooxygenase, MSA monooxygenase, MSAMO) is an enzyme with system name methanesulfonate,NADH:oxygen oxidoreductase.12 This enzyme catalyses the following chemical reaction

methanesulfonate + NADH + H+ + O2 ⇌ normal-⇌ \rightleftharpoons formaldehyde + NAD+ + sulfite + H2O

Methanesulfonate monooxygenase is a flavoprotein.
