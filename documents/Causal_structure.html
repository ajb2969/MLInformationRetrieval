Causal structure

In mathematical physics, the causal structure of a Lorentzian manifold describes the causal relationships between points in the manifold.

Introduction

In modern physics (especially general relativity) spacetime is represented by a Lorentzian manifold. The causal relations between points in the manifold are interpreted as describing which events in spacetime can influence which other events.

Minkowski spacetime is a simple example of a Lorentzian manifold. The causal relationships between points in Minkowski spacetime take a particularly simple form since the space is flat. See Causal structure of Minkowski spacetime for more information.

The causal structure of an arbitrary (possibly curved) Lorentzian manifold is made more complicated by the presence of curvature. Discussions of the causal structure for such manifolds must be phrased in terms of smooth curves joining pairs of points. Conditions on the tangent vectors of the curves then define the causal relationships.

Tangent vectors

If ( M , g ) M g \,(M,g) is a Lorentzian manifold (for metric g g g on manifold M M M ) then the tangent vectors at each point in the manifold can be classed into three different types. A tangent vector X X X is

  • timelike if g ( X , X ) > 0 g X X 0 \,g(X,X)>0
  • null if g ( X , X ) = 0 g X X 0 \,g(X,X)=0
  • spacelike if g ( X , X ) < 0 g X X 0 \,g(X,X)<0

(Here we use the ( + , - , - , - , ⋯ ) normal-⋯ (+,-,-,-,\cdots) metric signature). A tangent vector is called "non-spacelike" if it is null or timelike.

These names come from the simpler case of Minkowski spacetime (see Causal structure of Minkowski spacetime).

Time-orientability

At each point in M M M the timelike tangent vectors in the point's tangent space can be divided into two classes. To do this we first define an equivalence relation on pairs of timelike tangent vectors.

If X X X and Y Y Y are two timelike tangent vectors at a point we say that X X X and Y Y Y are equivalent (written X ∼ Y similar-to X Y X\sim Y ) if g ( X , Y ) > 0 g X Y 0 \,g(X,Y)>0 .

There are then two equivalence classes which between them contain all timelike tangent vectors at the point. We can (arbitrarily) call one of these equivalence classes "future-directed" and call the other "past-directed". Physically this designation of the two classes of future- and past-directed timelike vectors corresponds to a choice of an arrow of time at the point. The future- and past-directed designations can be extended to null vectors at a point by continuity.

A Lorentzian manifold is time-orientable1 if a continuous designation of future-directed and past-directed for non-spacelike vectors can be made over the entire manifold.

Curves

A path in M M M is a continuous map μ : Σ → M normal-: μ normal-→ normal-Σ M \mu:\Sigma\to M where Σ normal-Σ \Sigma is a nondegenerate interval (i.e., a connected set containing more than one point) in ℝ ℝ \mathbb{R} . A smooth path has μ μ \mu differentiable an appropriate number of times (typically C ∞ superscript C C^{\infty} ), and a regular path has nonvanishing derivative.

A curve in M M M is the image of a path or, more properly, an equivalence class of path-images related by re-parametrisation, i.e. homeomorphisms or diffeomorphisms of Σ normal-Σ \Sigma . When M M M is time-orientable, the curve is oriented if the parameter change is required to be monotonic.

Smooth regular curves (or paths) in M M M can be classified depending on their tangent vectors. Such a curve is

  • chronological (or timelike) if the tangent vector is timelike at all points in the curve.
  • null if the tangent vector is null at all points in the curve.
  • spacelike if the tangent vector is spacelike at all points in the curve.
  • causal (or non-spacelike) if the tangent vector is timelike or null at all points in the curve.

The requirements of regularity and nondegeneracy of Σ normal-Σ \Sigma ensure that closed causal curves (such as those consisting of a single point) are not automatically admitted by all spacetimes.

If the manifold is time-orientable then the non-spacelike curves can further be classified depending on their orientation with respect to time.

A chronological, null or causal curve in M M M is

  • future-directed if, for every point in the curve, the tangent vector is future-directed.
  • past-directed if, for every point in the curve, the tangent vector is past-directed.

These definitions only apply to causal (chronological or null) curves because only timelike or null tangent vectors can be assigned an orientation with respect to time.

  • A closed timelike curve is a closed curve which is everywhere future-directed timelike (or everywhere past-directed timelike).
  • A closed null curve is a closed curve which is everywhere future-directed null (or everywhere past-directed null).
  • The holonomy of the ratio of the rate of change of the affine parameter around a closed null geodesic is the redshift factor.

Causal relations

There are two types of causal relations between points x x x and y y y in the manifold M M M .

  • x x x chronologically precedes y y y (often denoted x ≪ y much-less-than x y \,x\ll y ) if there exists a future-directed chronological (timelike) curve from x x x to y y y .
  • x x x strictly causally precedes y y y (often denoted x < y x y x ) if there exists a future-directed causal (non-spacelike) curve from x x x to y y y .
  • x x x causally precedes y y y (often denoted x ≺ y precedes x y x\prec y or x ≤ y x y x\leq y ) if x x x strictly causally precedes y y y or x = y x y x=y .
  • x x x horismos y y y 2 (often denoted x → y normal-→ x y x\to y or x ↗ y normal-↗ x y x\nearrow y ) if x ≺ y precedes x y x\prec y and x ≪̸ y not-much-less-than x y x\not\ll y .

These relations are transitive:3

  • x ≪ y much-less-than x y x\ll y , y ≪ z much-less-than y z y\ll z implies x ≪ z much-less-than x z x\ll z
  • x ≺ y precedes x y \,x\prec y , y ≺ z precedes y z \,y\prec z implies x ≺ z precedes x z \,x\prec z

and satisfy4

  • x ≪ y much-less-than x y x\ll y implies x ≺ y precedes x y x\prec y (this follows trivially from the definition)
  • x ≪ y much-less-than x y x\ll y , y ≺ z precedes y z y\prec z implies x ≪ z much-less-than x z x\ll z
  • x ≺ y precedes x y x\prec y , y ≪ z much-less-than y z y\ll z implies x ≪ z much-less-than x z x\ll z

  • The chronological past of x x x , denoted I - ( x ) superscript I x \,I^{-}(x) , as the set of all points y y y in M M M such that y y y chronologically precedes x x x :

I - ( x ) = { y ∈ M | y ≪ x } superscript I x conditional-set y M much-less-than y x \,I^{-}(x)=\{y\in M|y\ll x\}

We similarly define

  • The causal future (also called the absolute future) of x x x , denoted J + ( x ) superscript J x \,J^{+}(x) , as the set of all points y y y in M M M such that x x x causally precedes y y y :

J + ( x ) = { y ∈ M | x ≺ y } superscript J x conditional-set y M precedes x y \,J^{+}(x)=\{y\in M|x\prec y\}

  • The causal past (also called the absolute past) of x x x , denoted J - ( x ) superscript J x \,J^{-}(x) , as the set of all points y y y in M M M such that y y y causally precedes x x x :

J - ( x ) = { y ∈ M | y ≺ x } superscript J x conditional-set y M precedes y x \,J^{-}(x)=\{y\in M|y\prec x\}

Points contained in I + ( x ) superscript I x \,I^{+}(x) , for example, can be reached from x x x by a future-directed timelike curve. The point x x x can be reached, for example, from points contained in J - ( x ) superscript J x \,J^{-}(x) by a future-directed non-spacelike curve.

As a simple example, in Minkowski spacetime the set I + ( x ) superscript I x \,I^{+}(x) is the interior of the future light cone at x x x . The set J + ( x ) superscript J x \,J^{+}(x) is the full future light cone at x x x , including the cone itself.

Conformal geometry

Two metrics g g \,g and g ^ normal-^ g \hat{g} are conformally related7 if g ^ = Ω 2 g normal-^ g superscript normal-Ω 2 g \hat{g}=\Omega^{2}g for some real function Ω normal-Ω \Omega called the conformal factor. (See conformal map).

Looking at the definitions of which tangent vectors are timelike, null and spacelike we see they remain unchanged if we use g g \,g or g ^ . normal-^ g \hat{g}. As an example suppose X X X is a timelike tangent vector with respect to the g g \,g metric. This means that g ( X , X ) > 0 g X X 0 \,g(X,X)>0 . We then have that g ^ ( X , X ) = Ω 2 g ( X , X ) > 0 normal-^ g X X superscript normal-Ω 2 g X X 0 \hat{g}(X,X)=\Omega^{2}g(X,X)>0 so X X X is a timelike tangent vector with respect to the g ^ normal-^ g \hat{g} too.

It follows from this that the causal structure of a Lorentzian manifold is unaffected by a conformal transformation.
