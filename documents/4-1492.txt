A stochastic matrix is a square matrix of nonnegative real entries in which the sum of the entries in each column is 1. Thus, a doubly stochastic matrix is a square matrix of nonnegative real entries in which the sum of the entries in each row and the sum of the entries in each column is 1.  The inequality  Muirhead's inequality states that [ a ] ≤ [ b ] for all x such that x i ≥ 0 for all x i if and only if there is some doubly stochastic matrix P for which a = Pb .  The latter condition can be expressed in several equivalent ways; one of them is given below.  The proof makes use of the fact that every doubly stochastic matrix is a weighted average of permutation matrices ( Birkhoff-von Neumann theorem ).
