North-East lattice paths

A North-East (NE) lattice path is a lattice path in ℤ 2 superscript ℤ 2 \mathbb{Z}^{2} with steps in S = { ( 0 , 1 ) , ( 1 , 0 ) } S 0 1 1 0 S=\{(0,1),(1,0)\} . The ( 0 , 1 ) 0 1 (0,1) steps are called North steps and denoted by N N N 's; the ( 1 , 0 ) 1 0 (1,0) steps are called East steps and denoted by E E E 's.

NE lattice paths most commonly begin at the origin. This convention allows us to encode all the information about a NE lattice path L L L in a single permutation word. The length of the word gives us the number of steps of the lattice path, k k k . The order of the N N N 's and E E E 's communicates the sequence of L L L . Furthermore, the number of N N N 's and the number of E E E 's in the word determines the end point of L L L .

If the permutation word for a NE lattice path contains n n n N N N steps and e e e E E E steps, and if the path begins at the origin, then the path necessarily ends at ( e , n ) e n (e,n) . This follows because you have "walked" exactly n n n steps North and e e e steps East from ( 0 , 0 ) 0 0 (0,0) .
