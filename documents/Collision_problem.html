The r-to-1 collision problem is an important theoretical problem in complexity theory, quantum computing, and computational mathematics. Classical Solutions

Deterministic

Solving the 2-to-1 version deterministically requires n / 2 + 1 n 2 1 n/2+1 queries, and in general distinguishing r-to-1 functions from 1-to-1 functions requires n / r + 1 n r 1 n/r+1 queries.

This is a straightforward application of the pigeonhole principle: if a function is r-to-1, then after n / r + 1 n r 1 n/r+1 queries we are guaranteed to have found a collision. If a function is 1-to-1, then no collision exists. Thus, n / r + 1 n r 1 n/r+1 queries suffice. If we are unlucky, then the first n / r n r n/r queries could return distinct answers, so n / r + 1 n r 1 n/r+1 queries is also necessary.
