Sigma-ring

In mathematics, a nonempty collection of sets is called a σ-ring (pronounced sigma-ring) if it is closed under countable union and relative complementation.

Similar concepts

If the first property is weakened to closure under finite union (i.e., A ∪ B ∈ ℛ A B ℛ A\cup B\in\mathcal{R} whenever A , B ∈ ℛ A B ℛ A,B\in\mathcal{R} ) but not countable union, then ℛ ℛ \mathcal{R} is a ring but not a σ-ring.

Uses

σ-rings can be used instead of σ-fields (σ-algebras) in the development of measure and integration theory, if one does not wish to require that the universal set be measurable. Every σ-field is also a σ-ring, but a σ-ring need not be a σ-field.

A σ-ring ℛ ℛ \mathcal{R} that is a collection of subsets of X X X induces a σ-field for X X X . Define 𝒜 𝒜 \mathcal{A} to be the collection of all subsets of X X X that are elements of ℛ ℛ \mathcal{R} or whose complements are elements of ℛ ℛ \mathcal{R} . Then 𝒜 𝒜 \mathcal{A} is a σ-field over the set X X X . In fact 𝒜 𝒜 \mathcal{A} is the minimal σ-field containing ℛ ℛ \mathcal{R} since it must be contained in every σ-field containing ℛ ℛ \mathcal{R} .
