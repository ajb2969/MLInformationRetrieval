Leonardo number

The Leonardo numbers are a sequence of numbers given by the recurrence:

L ( n ) = { 1 if n = 0 1 if n = 1 L ( n - 1 ) + L ( n - 2 ) + 1 if n > 1 L n cases 1 if n 0 1 if n 1 L n 1 L n 2 1 if n 1 L(n)=\begin{cases}1&\mbox{if }n=0\\ 1&\mbox{if }n=1\\ L(n-1)+L(n-2)+1&\mbox{if }n>1\\ \end{cases}

Edsger W. Dijkstra1 used them as an integral part of his smoothsort algorithm,2 and also analyzed them in some detail.3

Computing a second-order recurrence relation recursively and without memoization requires L(n) computations for the n-th item of the series.
