   Harmonious set      Harmonious set   In mathematics , a harmonious set is a subset of a locally compact abelian group on which every weak character may be uniformly approximated by strong characters. Equivalently, a suitably defined dual set is relatively dense in the Pontryagin dual of the group. This notion was introduced by Yves Meyer in 1970 and later turned out to play an important role in the mathematical theory of quasicrystals . Some related concepts are model sets , Meyer sets , and cut-and-project sets . Properties   A subset of a harmonious set is harmonious.    If Î› is a harmonious set and F is a finite set then the set Î› + F is also harmonious.   The next two properties show that the notion of a harmonious set is nontrivial only when the ambient group is neither compact nor discrete.   A finite set Î› is always harmonious. If the group G is compact then, conversely, every harmonious set is finite.    If G is a discrete group then every set is harmonious.
