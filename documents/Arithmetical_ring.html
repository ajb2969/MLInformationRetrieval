Arithmetical ring

    𝔞 + ( 𝔟 ∩ 𝔠 ) = ( 𝔞 + 𝔟 ) ∩ ( 𝔞 + 𝔠 ) 𝔞 𝔟 𝔠 𝔞 𝔟 𝔞 𝔠 \mathfrak{a}+(\mathfrak{b}\cap\mathfrak{c})=(\mathfrak{a}+\mathfrak{b})\cap(% \mathfrak{a}+\mathfrak{c})

An arithmetical domain is the same thing as a Prüfer domain.
