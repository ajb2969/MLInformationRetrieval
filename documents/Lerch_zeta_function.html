Lerch zeta function

In mathematics, the Lerch zeta-function, sometimes called the Hurwitz–Lerch zeta-function, is a special function that generalizes the Hurwitz zeta-function and the polylogarithm. It is named after the Czech mathematician Mathias Lerch 1.

Definition

The Lerch zeta-function is given by

L ( λ , α , s ) = ∑ n = 0 ∞ exp ( 2 π i λ n ) ( n + α ) s . L λ α s superscript subscript n 0 2 π i λ n superscript n α s L(\lambda,\alpha,s)=\sum_{n=0}^{\infty}\frac{\exp(2\pi i\lambda n)}{(n+\alpha)% ^{s}}.

Special cases

The Hurwitz zeta-function is a special case, given by

ζ ( s , α ) = L ( 0 , α , s ) = Φ ( 1 , s , α ) . ζ s α L 0 α s normal-Φ 1 s α \,\zeta(s,\alpha)=L(0,\alpha,s)=\Phi(1,s,\alpha).

The polylogarithm is a special case of the Lerch Zeta, given by

Li s ( z ) = z Φ ( z , s , 1 ) . subscript Li s z z normal-Φ z s 1 \,\textrm{Li}_{s}(z)=z\Phi(z,s,1).

The Legendre chi function is a special case, given by

χ n ( z ) = 2 - n z Φ ( z 2 , n , 1 / 2 ) . subscript χ n z superscript 2 n z normal-Φ superscript z 2 n 1 2 \,\chi_{n}(z)=2^{-n}z\Phi(z^{2},n,1/2).

The Riemann zeta-function is given by

ζ ( s ) = Φ ( 1 , s , 1 ) . ζ s normal-Φ 1 s 1 \,\zeta(s)=\Phi(1,s,1).

The Dirichlet eta-function is given by

η ( s ) = Φ ( - 1 , s , 1 ) . η s normal-Φ 1 s 1 \,\eta(s)=\Phi(-1,s,1).
