   Gilbert–Varshamov bound      Gilbert–Varshamov bound   In coding theory , the Gilbert–Varshamov bound (due to Edgar Gilbert 1 and independently Rom Varshamov 2 ) is a limit on the parameters of a (not necessarily linear ) code . It is occasionally known as the Gilbert– Shannon –Varshamov bound (or the GSV bound ), but the name "Gilbert–Varshamov bound" is by far the most popular. Varshamov proved this bound by using the probabilistic method for linear code. For more about that proof, see: GV-linear-code .
