Borwein integral

In mathematics, a Borwein integral is an integral involving products of sinc(ax), where the sinc function is given by sinc(x) = sin(x)/x for x not equal to 0, and sinc(0) = 1.12 These integrals are notorious for exhibiting apparent patterns that eventually break down. An example is as follows:

However at the next step the obvious pattern fails:

∫ 0 ∞ sin ( x ) x sin ( x / 3 ) x / 3 ⋯ sin ( x / 15 ) x / 15 d x superscript subscript 0 x x x 3 x 3 normal-⋯ x 15 x 15 d x \displaystyle\int_{0}^{\infty}\frac{\sin(x)}{x}\frac{\sin(x/3)}{x/3}\cdots% \frac{\sin(x/15)}{x/15}\,dx

In general similar integrals have value π/2 whenever the numbers 3, 5, ... are replaced by positive real numbers such that the sum of their reciprocals is less than 1. In the example above, 1/3 + 1/5 + ... + 1/13  1.
