 code > span.in { color: #60a0b0; font-weight: bold; font-style: italic; } /* Information */      Smoothstep is a scalar interpolation function commonly used in computer graphics 1 2 and video game engines . 3 The function interpolates smoothly between two input values based on a third one that should be between the first two. The returned value is clamped between 0 and 1.  The slope of the smoothstep function tends toward zero at both edges. This makes it easy to create a sequence of transitions using smoothstep to interpolate each segment rather than using a more sophisticated or expensive interpolation technique.  As pointed out in MSDN and OpenGL documentation, smoothstep implements cubic Hermite interpolation after doing a clamp:       smoothstep   (  t  )    =    3   t  2    -   2   t  3          smoothstep  t       3   superscript  t  2      2   superscript  t  3       \operatorname{smoothstep}(t)=3t^{2}-2t^{3}     A C/C++ example implementation provided by AMD 4 follows.  float smoothstep( float edge0, float edge1, float x)
 { // Scale, bias and saturate x to 0..1 range x = clamp((x - edge0)/(edge1 - edge0), 0.0 , 1.0 ); // Evaluate polynomial  return x*x*( 3 - 2 *x);
