Special unitary group

In mathematics, the special unitary group of degree n n n , denoted S U ( n ) S U n SU(n) , is the Lie group of n √ó n n normal-√ó n n√ón unitary matrices with determinant 1 (i.e., real-valued determinant, not complex as for general unitary matrices). The group operation is that of matrix multiplication. The special unitary group is a subgroup of the unitary group U ( n ) U n U(n) , consisting of all n √ó n n normal-√ó n n√ón unitary matrices. As a compact classical group, U ( n ) U n U(n) is the group that preserves the standard inner product on .1 It is itself a subgroup of the general linear group, S U ( n ) ‚äÇ U ( n ) ‚äÇ G L ( n , ùêÇ ) S U n normal-‚äÇ U n normal-‚äÇ G L n ùêÇ SU(n)‚äÇU(n)‚äÇGL(n,\mathbf{C}) .

The S U ( n ) S U n SU(n) groups find wide application in the Standard Model of particle physics, especially S U ( 2 ) S U 2 SU(2) in the electroweak interaction and S U ( 3 ) S U 3 SU(3) in quantum chromodynamics.2

The simplest case, S U ( 1 ) S U 1 SU(1) , is the trivial group, having only a single element. The group S U ( 2 ) S U 2 SU(2) is isomorphic to the group of quaternions of norm 1, and is thus diffeomorphic to the 3-sphere. Since unit quaternions can be used to represent rotations in 3-dimensional space (up to sign), there is a surjective homomorphism from S U ( 2 ) S U 2 SU(2) to the rotation group S O ( 3 ) S O 3 SO(3) whose kernel is }.3 S U ( 2 ) S U 2 SU(2) is also identical to one of the symmetry groups of spinors, Spin(3), that enables a spinor presentation of rotations.
