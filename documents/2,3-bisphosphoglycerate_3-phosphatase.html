2,3-bisphosphoglycerate 3-phosphatase

2,3-bisphosphoglycerate 3-phosphatase (, MIPP1, 2,3-BPG 3-phosphatase) is an enzyme with system name 2,3-bisphospho-D-glycerate 3-phosphohydrolase.1 This enzyme catalyses the following chemical reaction

2,3-bisphospho-D-glycerate + H2O $\rightleftharpoons$ 2-phospho-D-glycerate + phosphate

This reaction is a shortcut in the Rapoport-Luebering shunt.
