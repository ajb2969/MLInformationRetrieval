   Relative homology      Relative homology   In algebraic topology , a branch of mathematics , the (singular) homology of a topological space relative to a subspace is a construction in singular homology , for pairs of spaces . The relative homology is useful and important in several ways. Intuitively, it helps determine what part of an absolute homology group comes from which subspace. It follows that H n ( X , x 0 ), where x 0 is a point in X , is the n -th reduced homology group of X . In other words, H i ( X , x 0 ) = H i ( X ) for all i > 0. When i = 0, H 0 ( X , x 0 ) is the free module of one rank less than H 0 ( X ). The connected component containing x 0 becomes trivial in relative homology. The n -th local homology group of a space X at a point x 0 is defined to be H n ( X , X - { x 0 }). Informally, this is the "local" homology of X close to x 0 .
