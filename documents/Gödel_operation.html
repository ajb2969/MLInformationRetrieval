Gödel operation

In mathematical set theory, a set of Gödel operations is a finite collection of operations on sets that can be used to construct the constructible sets from ordinals. introduced the original set of 8 Gödel operations 𝔉1,...,𝔉8 under the name fundamental operations. Other authors sometimes use a slightly different set of about 8 to 10 operations, usually denoted G1, G2,...

Definition

used the following eight operations as a set of Gödel operations (which he called fundamental operations):

  1. 𝔉 1 ( X , Y ) = { X , Y } subscript 𝔉 1 X Y X Y \mathfrak{F}_{1}(X,Y)=\{X,Y\}
  2. 𝔉 2 ( X , Y ) = E ⋅ X = { ( a , b ) \isin X ∣ a \isin b } subscript 𝔉 2 X Y normal-⋅ E X conditional-set a b \isin X a \isin b \mathfrak{F}_{2}(X,Y)=E\cdot X=\{(a,b)\isin X\mid a\isin b\}
  3. 𝔉 3 ( X , Y ) = X - Y subscript 𝔉 3 X Y X Y \mathfrak{F}_{3}(X,Y)=X-Y
  4. 𝔉 4 ( X , Y ) = X ↾ Y = X ⋅ ( V × Y ) = { ( a , b ) \isin X ∣ b \isin Y } subscript 𝔉 4 X Y X normal-↾ Y normal-⋅ X V Y conditional-set a b \isin X b \isin Y \mathfrak{F}_{4}(X,Y)=X\upharpoonright Y=X\cdot(V\times Y)=\{(a,b)\isin X\mid b% \isin Y\}
  5. 𝔉 5 ( X , Y ) = X ⋅ 𝔇 ( Y ) = { b \isin X ∣ ∃ a ( a , b ) \isin Y } subscript 𝔉 5 X Y normal-⋅ X 𝔇 Y conditional-set b \isin X a a b \isin Y \mathfrak{F}_{5}(X,Y)=X\cdot\mathfrak{D}(Y)=\{b\isin X\mid\exists a(a,b)\isin Y\}
  6. 𝔉 6 ( X , Y ) = X ⋅ Y - 1 = { ( a , b ) \isin X ∣ ( b , a ) \isin Y } subscript 𝔉 6 X Y normal-⋅ X superscript Y 1 conditional-set a b \isin X b a \isin Y \mathfrak{F}_{6}(X,Y)=X\cdot Y^{-1}=\{(a,b)\isin X\mid(b,a)\isin Y\}
  7. 𝔉 7 ( X , Y ) = X ⋅ ℭ 𝔫 𝔳 2 ( Y ) = { ( a , b , c ) \isin X ∣ ( a , c , b ) \isin Y } subscript 𝔉 7 X Y normal-⋅ X ℭ 𝔫 subscript 𝔳 2 Y conditional-set a b c \isin X a c b \isin Y \mathfrak{F}_{7}(X,Y)=X\cdot\mathfrak{Cnv}_{2}(Y)=\{(a,b,c)\isin X\mid(a,c,b)% \isin Y\}
  8. 𝔉 8 ( X , Y ) = X ⋅ ℭ 𝔫 𝔳 3 ( Y ) = { ( a , b , c ) \isin X ∣ ( c , a , b ) \isin Y } subscript 𝔉 8 X Y normal-⋅ X ℭ 𝔫 subscript 𝔳 3 Y conditional-set a b c \isin X c a b \isin Y \mathfrak{F}_{8}(X,Y)=X\cdot\mathfrak{Cnv}_{3}(Y)=\{(a,b,c)\isin X\mid(c,a,b)% \isin Y\}

The second expression in each line gives Gödel's definition in his original notation, where the dot means intersection, V is the universe, E is the membership relation, and so on.

uses the following set of 10 Gödel operations.

  1. G 1 ( X , Y ) = { X , Y } subscript G 1 X Y X Y G_{1}(X,Y)=\{X,Y\}
  2. G 2 ( X , Y ) = X × Y subscript G 2 X Y X Y G_{2}(X,Y)=X\times Y
  3. G 3 ( X , Y ) = { ( x , y ) ∣ x \isin X , y \isin Y , x \isin y } subscript G 3 X Y conditional-set x y x \isin X y \isin Y x \isin y G_{3}(X,Y)=\{(x,y)\mid x\isin X,y\isin Y,x\isin y\}
  4. G 4 ( X , Y ) = X - Y subscript G 4 X Y X Y G_{4}(X,Y)=X-Y
  5. G 5 ( X , Y ) = X ∩ Y subscript G 5 X Y X Y G_{5}(X,Y)=X\cap Y
  6. G 6 ( X ) = ∪ X subscript G 6 X X G_{6}(X)=\cup X
  7. G 7 ( X ) = dom ( X ) subscript G 7 X dom X G_{7}(X)=\text{dom}(X)
  8. G 8 ( X ) = { ( x , y ) ∣ ( y , x ) \isin X } subscript G 8 X conditional-set x y y x \isin X G_{8}(X)=\{(x,y)\mid(y,x)\isin X\}
  9. G 9 ( X ) = { ( x , y , z ) ∣ ( x , z , y ) \isin X } subscript G 9 X conditional-set x y z x z y \isin X G_{9}(X)=\{(x,y,z)\mid(x,z,y)\isin X\}
  10. G 10 ( X ) = { ( x , y , z ) ∣ ( y , z , x ) \isin X } subscript G 10 X conditional-set x y z y z x \isin X G_{10}(X)=\{(x,y,z)\mid(y,z,x)\isin X\}

Properties

Gödel's normal form theorem states that if φ(x1,...xn) is a formula with all quantifiers bounded, then the function {(x1,...,xn) ∈ X1×...×Xn | φ(x1, ..., xn)) of X1, ..., Xn is given by a composition of some Gödel operations.
