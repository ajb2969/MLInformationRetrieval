   Ordinal regression      Ordinal regression   In statistics , ordinal regression is a type of regression analysis used for predicting an ordinal variable , i.e. a variable whose value exists on an arbitrary scale where only the relative ordering between different values is significant. It can be considered an intermediate problem in between (metric) regression and classification . 1 Ordinal regression turns up often in the social sciences , for example in the modeling of human levels of preference (on a scale from, say, 1â€“5 for "very poor" through "excellent"), as well as in information retrieval . In machine learning , ordinal regression may also be called ranking learning . 2  Linear models for ordinal regression  Ordinal regression can be performed using a generalized linear model (GLM) that fits both a coefficient vector and a set of thresholds to a dataset.
