Applications

One application of ‚Ñì‚àû and L ‚àû subscript L L_{\infty} is in economies with infinitely many commodities.1 In simple economic models, it is common to assume that there is only a finite number of different commodities, e.g. houses, fruits, cars, etc., so every bundle can be represented by a finite vector, and the consumption set is a vector space with a finite dimension.
