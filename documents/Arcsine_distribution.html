In probability theory, the arcsine distribution is the probability distribution whose cumulative distribution function is

F ( x ) = 2 π arcsin ( x ) = arcsin ( 2 x - 1 ) π + 1 2 F x 2 π x 2 x 1 π 1 2 F(x)=\frac{2}{\pi}\arcsin\left(\sqrt{x}\right)=\frac{\arcsin(2x-1)}{\pi}+\frac% {1}{2}

for 0 ≤ x ≤ 1, and whose probability density function is

f ( x ) = 1 π x ( 1 - x ) f x 1 π x 1 x f(x)=\frac{1}{\pi\sqrt{x(1-x)}}

on (0, 1). The standard arcsine distribution is a special case of the beta distribution with α = β = 1/2. That is, if X X X is the standard arcsine distribution then X ∼ Beta ( 1 2 , 1 2 ) similar-to X Beta 1 2 1 2 X\sim{\rm Beta}(\tfrac{1}{2},\tfrac{1}{2})

The arcsine distribution appears

  • in the Lévy arcsine law;
  • in the Erdős arcsine law;
  • as the Jeffreys prior for the probability of success of a Bernoulli trial.

Arbitrary bounded support

The distribution can be expanded to include any bounded support from a ≤ x ≤ b by a simple transformation

F ( x ) = 2 π arcsin ( x - a b - a ) F x 2 π x a b a F(x)=\frac{2}{\pi}\arcsin\left(\sqrt{\frac{x-a}{b-a}}\right)

for a ≤ x ≤ b, and whose probability density function is

f ( x ) = 1 π ( x - a ) ( b - x ) f x 1 π x a b x f(x)=\frac{1}{\pi\sqrt{(x-a)(b-x)}}

on (a, b).
