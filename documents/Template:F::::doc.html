Usage

This template produces the notation for f-number, using a hooked as is common in photography. In more mathematical articles, it may be preferable to code the symbol using the markup used for other math in the article: ''f''/ = f/, or f/ = f / fragments f f/ .

Usage:

  • "{{f/}}2" produces 2
  • "{{f/|2}}" produces
  • "{{f/|2|link=yes}}" produces
  • "{{f/|#}}" produces
