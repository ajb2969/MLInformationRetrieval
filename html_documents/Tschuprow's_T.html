In statistics, Tschuprow's T is a measure of association between two nominal variables, giving a value between 0 and 1 (inclusive). It is closely related to Cramér's V, coinciding with it for square contingency tables. It was published by Alexander Tschuprow (alternative spelling: Chuprov) in 1939.1

Properties

T equals zero if and only if independence holds in the table, i.e., if and only if π i j = π i + π + j subscript π i j subscript π limit-from i subscript π j \pi_{ij}=\pi_{i+}\pi_{+j} . T equals one if and only there is perfect dependence in the table, i.e., if and only if for each i there is only one j such that π i j > 0 subscript π i j 0 \pi_{ij}>0 and vice versa. Hence, it can only equal 1 for square tables. In this it differs from Cramér's V, which can be equal to 1 for any rectangular table.
