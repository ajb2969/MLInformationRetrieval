Definition

The test is applicable to testing the agreement between two observers. It is defined to be

B = Σ n i i 2 Σ n i . n . i B normal-Σ superscript subscript n i i 2 normal-Σ subscript n fragments i normal-. subscript n fragments normal-. i B=\frac{\Sigma n_{ii}^{2}}{\Sigma n_{i.}n_{.i}}

where nii are the values on the main diagonal. B varies in value between 0 (no agreement) and +1 (perfect agreement).

In large samples B has a normal distribution whose variance has a complicated expression.4 For small samples a permutation test is indicated.5

Guidance on its use and its extension to n x n tables have been provided by Munoz & Bangdiwala.6 It may be more useful than the more commonly used Cohen's kappa in some circumstances.7

Tutorials and examples

Worked examples of the use of Bangdiwala's B have been published.89 The statistical programming language R has a set of functions that will compute the B test,10 and a tutorial on the use of a test using these R functions is available.11
