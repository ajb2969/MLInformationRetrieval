Torulene dioxygenase

Torulene dioxygenase (, CAO-2, CarT) is an enzyme with system name torulene:oxygen oxidoreductase.123 This enzyme catalyses the following chemical reaction

torulene + O2 ⇌ normal-⇌ \rightleftharpoons 4'-apo-beta,psi-caroten-4'-al + 3-methylbut-2-enal

It is assumed that 3-methylbut-2-enal is formed.
