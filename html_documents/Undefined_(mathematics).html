Undefined (mathematics)

In mathematics, the word undefined has several different meanings, depending on the context.

  • In geometry, simple words such as "point" and "line" are taken as undefined terms.
  • In arithmetic, some arithmetic operations are called "undefined". Two of the more famous examples are division by zero and zero to the power of zero.
  • In algebra, a function is said to be "undefined" at points not in its domain. For example, in the real number system, f ( x ) = x f x x f(x)=\sqrt{x} is undefined for negative  x x x , i.e., no such values exist for function f f f .

Undefined terms in geometry

In ancient times, geometers attempted to define every term. For example, Euclid defined a point as "that which has no part". In modern times, mathematicians recognized that attempting to define every word inevitably led to circular definitions, and therefore left some terms, "point" for example, as undefined (see primitive notion).

Indeterminate operations in arithmetic

The expression 0/0 is indeterminate in arithmetic, as explained in division by zero (the expression is used in calculus to represent an indeterminate form).

00 is often left indeterminate, see zero to the power of zero for details.

Values for which functions are undefined

The set of numbers for which a function is defined is called the domain of the function. If a number is not in the domain of a function, the function is said to be "undefined" for that number. Two common examples are f ( x ) = 1 x f x 1 x f(x)=\frac{1}{x} , which is undefined for x = 0 x 0 x=0 , and f ( x ) = x f x x f(x)=\sqrt{x} , which is undefined (in the real number system) for negative  x x x .

Notation using ↓ and ↑

In computability theory, if f is a partial function on S and a is an element of S, then this is written as f(a)↓ and is read as "f(a) is defined." 1

If a is not in the domain of f, then this is written as f(a)↑ and is read as "f(a) is undefined".

Singularities in complex analysis

In complex analysis, a point z ∈ ℂ z ℂ z\in\mathbb{C} where a holomorphic function is undefined is called a singularity. One distinguishes between removable singularities (the function can be extended holomorphically to z z z , poles (the function can be extended meromorphically to z z z ), and essential singularities, where no meromorphic extension to z z z exists.
