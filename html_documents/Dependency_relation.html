Dependency relation

In mathematics and computer science, a dependency relation is a binary relation that is finite, symmetric, and reflexive; i.e. a finite tolerance relation. That is, it is a finite set of ordered pairs D D D , such that

  • If ( a , b ) ∈ D a b D (a,b)\in D then ( b , a ) ∈ D b a D (b,a)\in D (symmetric)
  • If a a a is an element of the set on which the relation is defined, then ( a , a ) ∈ D a a D (a,a)\in D (reflexive)

In general, dependency relations are not transitive; thus, they generalize the notion of an equivalence relation by discarding transitivity.
