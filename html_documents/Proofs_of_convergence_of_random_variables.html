Convergence in probability does not imply almost sure convergence in the discrete case

If Xn are independent random variables assuming value one with probability 1/n and zero otherwise, then Xn converges to zero in probability but not almost surely. This can be verified using the Borel‚ÄìCantelli lemmas.
