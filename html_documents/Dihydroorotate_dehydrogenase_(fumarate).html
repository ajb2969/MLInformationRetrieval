Dihydroorotate dehydrogenase (fumarate)

Dihydroorotate dehydrogenase (fumarate) (, dihydroorotate oxidase, pyr4 (gene)) is an enzyme with system name (S)-dihydroorotate:fumarate oxidoreductase.123456 This enzyme catalyses the following chemical reaction

(S)-dihydroorotate + fumarate ⇌ normal-⇌ \rightleftharpoons orotate + succinate

This enzyme contains FMN.
