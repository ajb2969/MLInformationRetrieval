CTP-dependent riboflavin kinase

CTP-dependent riboflavin kinase (, Methanocaldococcus jannaschii Mj0056, Mj0056) is an enzyme with system name CTP:riboflavin 5′-phosphotransferase.1 This enzyme catalyses the following chemical reaction

CTP + riboflavin ⇌ normal-⇌ \rightleftharpoons CDP + FMN

This archaeal enzyme uses CTP as the donor nucleotide.
