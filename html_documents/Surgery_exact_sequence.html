Surgery exact sequence

In the mathematical surgery theory the surgery exact sequence is the main technical tool to calculate the surgery structure set of a compact manifold in dimension > 4 absent 4 >4 . The surgery structure set ùíÆ ( X ) ùíÆ X \mathcal{S}(X) of a compact n n n -dimensional manifold X X X is a pointed set which classifies n n n -dimensional manifolds within the homotopy type of X X X .

The basic idea is that in order to calculate ùíÆ ( X ) ùíÆ X \mathcal{S}(X) it is enough to understand the other terms in the sequence, which are usually easier to determine. These are on one hand the normal invariants which form generalized cohomology groups, and hence one can use standard tools of algebraic topology to calculate them at least in principle. On the other hand there are the L-groups which are defined algebraically in terms of quadratic forms or in terms of chain complexes with quadratic structure. A great deal is known about these groups. Another part of the sequence are the surgery obstruction maps from normal invariants to the L-groups. For these maps there are certain characteristic classes formulas, which enable to calculate them in some cases. Knowledge of these three components, that means: the normal maps, the L-groups and the surgery obstruction maps is enough to determine the structure set (at least up to extension problems).

In practice one has to proceed case by case, for each manifold X X \mathcal{}X it is a unique task to determine the surgery exact sequence, see some examples below. Also note that there are versions of the surgery exact sequence depending on the category of manifolds we work with: smooth (DIFF), PL, or topological manifolds and whether we take Whitehead torsion into account or not (decorations s s s or h h h ).

The original 1962 work of Browder and Novikov on the existence and uniqueness of manifolds within a simply-connected homotopy type was reformulated by Sullivan in 1966 as a surgery exact sequence. In 1970 Wall developed non-simply-connected surgery theory and the surgery exact sequence for manifolds with arbitrary fundamental group.

Versions revisited

In the topological category the surgery obstruction map can be made into a homomorphism. This is achieved by putting an alternative abelian group structure on the normal invariants as described here. Moreover, the surgery exact sequence can be identified with the algebraic surgery exact sequence of Ranicki which is an exact sequence of abelian groups by definition. This gives the structure set ùíÆ ( X ) ùíÆ X \mathcal{S}(X) the structure of an abelian group. Note, however, that there is to this date no satisfactory geometric description of this abelian group structure.
