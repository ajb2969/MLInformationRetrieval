Glutaryl-CoA dehydrogenase (non-decarboxylating)

Glutaryl-CoA dehydrogenase (non-decarboxylating) (, GDHDes, nondecarboxylating glutaryl-coenzyme A dehydrogenase, nondecarboxylating glutaconyl-coenzyme A-forming GDH) is an enzyme with system name glutaryl-CoA:acceptor 2,3-oxidoreductase (non-decarboxylating).12 This enzyme catalyses the following chemical reaction

glutaryl-CoA + acceptor ⇌ normal-⇌ \rightleftharpoons (E)-glutaconyl-CoA + reduced acceptor

The enzyme contains FAD.
