Final value theorem

In mathematical analysis, the final value theorem (FVT) is one of several similar theorems used to relate frequency domain expressions to the time domain behavior as time approaches infinity. A final value theorem allows the time domain behavior to be directly calculated by taking a limit of a frequency domain expression, as opposed to converting to a time domain expression and taking its limit.

Example where FVT does not hold

However, for a system described by the transfer function

H ( s ) = 9 s 2 + 9 , H s 9 superscript s 2 9 H(s)=\frac{9}{s^{2}+9}, the final value theorem appears to predict the final value of the impulse response to be 0 and the final value of the step response to be 1. However, neither time-domain limit exists, and so the final value theorem predictions are not valid. In fact, both the impulse response and step response oscillate, and (in this special case) the final value theorem describes the average values around which the responses oscillate.
