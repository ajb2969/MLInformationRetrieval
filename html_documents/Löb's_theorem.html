Löb's theorem

In mathematical logic, Löb's theorem states that in a theory with Peano arithmetic, for any formula P, if it is provable that "if P is provable then P is true", then P is provable. I.e.

if P A ⊢ ( B e w ( # P ) → P ) , then P A ⊢ P fragments if P A proves fragments normal-( B e w fragments normal-( # P normal-) normal-→ P normal-) normal-, then P A proves P \mathrm{if}\ PA\vdash(Bew(\#P)\rightarrow P)\mathrm{,then}\ PA\vdash P

where Bew(#P) means that the formula P with Gödel number #P is provable.

Löb's theorem is named for Martin Hugo Löb, who formulated it in 1955.

Modal Proof of Löb's theorem

Löb's theorem can be proved within modal logic using only some basic rules about the provability operator (the K4 system) plus the existence of modal fixed points.

Modal Formulas

We will assume the following grammar for formulas:

  1. If X X X is a propositional variable, then X X X is a formula.
  2. If K K K is a propositional constant, then K K K is a formula.
  3. If A A A is a formula, then □ A normal-□ A \Box A is a formula.
  4. If A A A and B B B are formulas, then so are ¬ A A \neg A , A → B normal-→ A B A\rightarrow B , A ∧ B A B A\wedge B , A ∨ B A B A\vee B , and A ↔ B normal-↔ A B A\leftrightarrow B

A modal sentence is a modal formula that contains no propositional variables. We use ⊢ A proves absent A \vdash A to mean A A A is a theorem.

If F ( X ) F X F(X) is a modal formula with only one propositional variable X X X , then a modal fixed point of F ( X ) F X F(X) is a sentence Ψ normal-Ψ \Psi such that

⊢ Ψ ↔ F ( □ Ψ ) fragments proves Ψ normal-↔ F fragments normal-( □ Ψ normal-) \vdash\Psi\leftrightarrow F(\Box\Psi)

We will assume the existence of such fixed points for every modal formula with one free variable. This is of course not an obvious thing to assume, but if we interpret □ normal-□ \Box as provability in Peano Arithmetic, then the existence of modal fixed points is in fact true.

Modal Rules of Inference

In addition to the existence of modal fixed points, we assume the following rules of inference for the provability operator □ normal-□ \Box :

  1. From ⊢ A proves absent A \vdash A conclude ⊢ □ A proves absent normal-□ A \vdash\Box A : Informally, this says that if A is a theorem, then it is provable.
  2. ⊢ □ A → □ □ A proves absent normal-→ normal-□ A normal-□ normal-□ A \vdash\Box A\rightarrow\Box\Box A : If A is provable, then it is provable that it is provable.

If it is provable that A implies B, and A is provable, then B is provable.

More on the existence of modal fixed points

Not only does the existence of modal fixed points imply Löb's theorem, but the converse is valid, too. When Löb's theorem is given as an axiom (schema), the existence of a fixed point (up to provable equivalence) p ↔ A ( p ) normal-↔ p A p p\leftrightarrow A(p) for any formula A(p)'' modalized in p'' can be derived.1 Thus in normal modal logic, Löb's axiom is equivalent to the conjunction of the axiom schema 4, ( □ A → □ □ A , ) fragments normal-( □ A normal-→ □ □ A normal-, normal-) (\Box A\rightarrow\Box\Box A,) and the existence of modal fixed points.
