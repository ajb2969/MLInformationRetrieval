Atom (measure theory)

In mathematics, more precisely in measure theory, an atom is a measurable set which has positive measure and contains no set of smaller but positive measure. A measure which has no atoms is called non-atomic or atomless.

Definition

Given a measurable space ( X , Σ ) X normal-Σ (X,\Sigma) and a measure μ μ \mu on that space, a set A ⊂ X A X A\subset X in Σ normal-Σ \Sigma is called an atom if

μ ( A ) > 0 μ A 0 \mu(A)>0

and for any measurable subset B ⊂ A B A B\subset A with

μ ( B ) < μ ( A ) μ B μ A \mu(B)<\mu(A)

the set B B B has measure zero.

Examples

  • Consider the set X={1, 2, ..., 9, 10} and let the sigma-algebra Σ normal-Σ \Sigma be the power set of X. Define the measure μ μ \mu of a set to be its cardinality, that is, the number of elements in the set. Then, each of the singletons {i}, for i=1,2, ..., 9, 10 is an atom.
  • Consider the Lebesgue measure on the real line. This measure has no atoms.

there exists a measurable subset B of A such that

μ ( B ) = b . μ B b \mu(B)=b.\,

This theorem is due to Wacław Sierpiński.12 It is reminiscent of the intermediate value theorem for continuous functions.
