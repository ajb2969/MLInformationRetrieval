Finite Fourier transform

In mathematics the finite Fourier transform may refer to either

  • another name for the discrete Fourier transform1

or

  • another name for the Fourier series coefficients2

or

  • a transform based on a Fourier-transform-like integral applied to a function x ( t ) x t x(t) , but with integration only on a finite interval, usually taken to be the interval [ 0 , T ] 0 T [0,T] .3 Equivalently, it is the Fourier transform of a function x ( t ) x t x(t) multiplied by a rectangular window function.
