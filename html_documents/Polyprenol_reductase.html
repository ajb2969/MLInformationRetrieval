Polyprenol reductase

Polyprenol reductase (, SRD5A3 (gene), DFG10 (gene)) is an enzyme with system name ditrans,polycis-dolichol:NADP+ 2,3-oxidoreductase.12 This enzyme catalyses the following chemical reaction

ditrans,polycis-dolichol + NADP+ ⇌ normal-⇌ \rightleftharpoons ditrans,polycis-polyprenol + NADPH + H+

The reaction occurs in the reverse direction.
