Pentalenene oxygenase

Pentalenene oxygenase (, PtlI) is an enzyme with system name pentalenene,NADPH:oxygen 13-oxidoreductase.1 This enzyme catalyses the following chemical reaction

pentalenene + 2 NADPH + 2 H+ + 2 O2 ⇌ normal-⇌ \rightleftharpoons pentalen-13-al + 2 NADP+ + 3 H2O (overall reaction)
(1a) pentalenene + NADPH + H+ + O2 ⇌ normal-⇌ \rightleftharpoons pentalen-13-ol + NADP+ + H2O
(1b) pentalen-13-ol + NADPH + H+ + O2 ⇌ normal-⇌ \rightleftharpoons pentalen-13-al + NADP+ + 2 H2O

Pentalenene oxygenase is a heme-thiolate protein (P-450).
