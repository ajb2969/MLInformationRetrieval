FMN reductase (NADPH)

FMN reductase (NADPH) (, FRP, flavin reductase P, SsuE) is an enzyme with system name FMNH2:NADP+ oxidoreductase.12345678 This enzyme catalyses the following chemical reaction:

FMNH2 + NADP+ $\rightleftharpoons$ FMN + NADPH + H+

The enzymes from bioluminescent bacteria contain FMN.
