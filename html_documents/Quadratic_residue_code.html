Adding an overall parity-check digit to a quadratic residue code gives an extended quadratic residue code. When p ≡ 3 p 3 p\equiv 3 (mod 4 4 4 ) an extended quadratic residue code is self-dual; otherwise it is equivalent but not equal to its dual.
