HOL Light

HOL Light is a member of the HOL theorem prover family. Like the other members, it is a proof assistant for classical higher order logic. Compared with other HOL systems, HOL Light is intended to have relatively simple foundations. HOL Light is authored and maintained by the mathematician and computer scientist John Harrison. HOL Light is released under the simplified BSD license.1

Logical foundations

HOL Light is based on a formulation of type theory with equality as the only primitive notion. The primitive rules of inference are the following:

⊢ t = t continued-fraction absent proves absent t t \cfrac{\qquad}{\vdash t=t}

REFL

reflexivity of equality

Γ ⊢ s = t Δ ⊢ t = u Γ ∪ Δ ⊢ s = u continued-fraction proves normal-Γ s t normal-Δ proves t u proves normal-Γ normal-Δ s u \cfrac{\Gamma\vdash s=t\qquad\Delta\vdash t=u}{\Gamma\cup\Delta\vdash s=u}

TRANS

transitivity of equality

Γ ⊢ f = g Δ ⊢ x = y Γ ∪ Δ ⊢ f ( x ) = g ( y ) continued-fraction proves normal-Γ f g normal-Δ proves x y proves normal-Γ normal-Δ f x g y \cfrac{\Gamma\vdash f=g\qquad\Delta\vdash x=y}{\Gamma\cup\Delta\vdash f(x)=g(y)}

MK_COMB

congruence of equality

Γ ⊢ s = t Γ ⊢ ( λ x . s ) = ( λ x . t ) continued-fraction proves normal-Γ s t fragments Γ proves fragments normal-( λ x normal-. s normal-) fragments normal-( λ x normal-. t normal-) \cfrac{\Gamma\vdash s=t}{\Gamma\vdash(\lambda x.s)=(\lambda x.t)}

ABS

abstraction of equality ( x x x must not be free in Γ normal-Γ \Gamma )

⊢ ( λ x . t ) x = t continued-fraction absent fragments proves fragments normal-( λ x normal-. t normal-) x t \cfrac{\qquad}{\vdash(\lambda x.t)x=t}

BETA

connection of abstraction and function application

{ p } ⊢ p continued-fraction absent proves p p \cfrac{\qquad}{\{p\}\vdash p}

ASSUME

assuming p p p , prove p p p

Γ ⊢ p = q Δ ⊢ p Γ ∪ Δ ⊢ q continued-fraction proves normal-Γ p q normal-Δ proves p proves normal-Γ normal-Δ q \cfrac{\Gamma\vdash p=q\qquad\Delta\vdash p}{\Gamma\cup\Delta\vdash q}

EQ_MP

relation of equality and deduction

Γ ⊢ p Δ ⊢ q ( Γ - { q } ) ∪ ( Δ - { p } ) ⊢ p = q continued-fraction proves normal-Γ p normal-Δ proves q proves normal-Γ q normal-Δ p p q \cfrac{\Gamma\vdash p\qquad\Delta\vdash q}{(\Gamma-\{q\})\cup(\Delta-\{p\})% \vdash p=q}

DEDUCT_ANTISYM_RULE

deduce equality from 2-way deducibility

Γ [ x 1 , … , x n ] ⊢ p [ x 1 , … , x n ] Γ [ t 1 , … , t n ] ⊢ p [ t 1 , … , t n ] continued-fraction proves normal-Γ subscript x 1 normal-… subscript x n p subscript x 1 normal-… subscript x n proves normal-Γ subscript t 1 normal-… subscript t n p subscript t 1 normal-… subscript t n \cfrac{\Gamma[x_{1},\ldots,x_{n}]\vdash p[x_{1},\ldots,x_{n}]}{\Gamma[t_{1},% \ldots,t_{n}]\vdash p[t_{1},\ldots,t_{n}]}

INST

instantiate variables in assumptions and conclusion of theorem

Γ [ α 1 , … , α n ] ⊢ p [ α 1 , … , α n ] Γ [ τ 1 , … , τ n ] ⊢ p [ τ 1 , … , τ n ] continued-fraction proves normal-Γ subscript α 1 normal-… subscript α n p subscript α 1 normal-… subscript α n proves normal-Γ subscript τ 1 normal-… subscript τ n p subscript τ 1 normal-… subscript τ n \cfrac{\Gamma[\alpha_{1},\ldots,\alpha_{n}]\vdash p[\alpha_{1},\ldots,\alpha_{% n}]}{\Gamma[\tau_{1},\ldots,\tau_{n}]\vdash p[\tau_{1},\ldots,\tau_{n}]}
