The graph was created with matlab, using the following script:

D = [-1:0.1:1];
[X Y] = meshgrid(D,D);
L=sqrt(X.^2+Y.^2);
A=atan(Y./X);
A(isnan(A))=0;
Z=L.*sin(2*A);
surfc(X,Y,Z);
view(15,40)
xlabel('x')
ylabel('y')
zlabel('f(x,y)')
axis equal
camlight headlight
lighting gouraud
print -dpng f.png
