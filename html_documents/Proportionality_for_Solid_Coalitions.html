Proportionality for Solid Coalitions

Proportionality for Solid Coalitions (PSC) is a voting system criterion relating to ranked voting systems. It's the essential requirement1 to guarantee a proportional representation of voters in multiple winner ranked voting systems.

Solid Coalitions

A set of voters V V V is a solid coalition for a set of candidates C C C , if every voter in V V V ranks every candidate in C C C ahead of every candidate that is not in C C C .

In the following let n n n be the number of voters, k k k be the number of seats to be filled and j j j be some positive integer.

k k k –PSC

k k k –PSC is defined with respect to the Hare quota n / k n k n/k . If V V V is a solid coalition for C C C and the number of Voters in V V V is at least j j j Hare quotas, then at least j j j candidates from C C C must be elected (if C C C has less than j j j candidates at all, then all of them have to be elected).2 This criterion was proposed by Michael Dummett.3

k + 1 k 1 k+1 –PSC

k + 1 k 1 k+1 –PSC is defined like k k k –PSC, but with respect to the Hagenbach-Bischoff quota n / ( k + 1 ) n k 1 n/(k+1) instead of the Hare quota: the number of voters in V V V must exceed j j j Hagenbach-Bischoff quotas.4 It is a generalization of the majority criterion in the sense that it relates to groups of supported candidates (solid coalitions) instead of just one candidate, and there may be more than one seat to be filled. Because some authors call the fraction n / ( k + 1 ) n k 1 n/(k+1) Droop Quota, k + 1 k 1 k+1 –PSC is also known as Droop proportionality criterion.5
