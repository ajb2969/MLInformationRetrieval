Range (mathematics)

In mathematics, and more specifically in naive set theory, the range of a function refers to either the codomain or the image of the function, depending upon usage. Modern usage almost always uses range to mean image.

The codomain of a function is some arbitrary set. In real analysis, it is the real numbers. In complex analysis, it is the complex numbers.

The image of a function is the set of all outputs of the function. The image is always a subset of the codomain.

Distinguishing between the two uses

As the term "range" can have different meanings, it is considered a good practice to define it the first time it is used in a textbook or article.

Older books, when they use the word "range", tend to use it to mean what is now called the codomain.12 More modern books, if they use the word "range" at all, generally use it to mean what is now called the image.3 To avoid any confusion, a number of modern books don't use the word "range" at all.4

As an example of the two different usages, consider the function f ( x ) = x 2 f x superscript x 2 f(x)=x^{2} as it is used in real analysis, that is, as a function that inputs a real number and outputs its square. In this case, its codomain is the set of real numbers ℝ ℝ \mathbb{R} , but its image is the set of non-negative real numbers ℝ + superscript ℝ \mathbb{R}^{+} , since x 2 superscript x 2 x^{2} is never negative if x x x is real. For this function, if we use "range" to mean codomain, it refers to ℝ ℝ \mathbb{R} . When we use "range" to mean image, it refers to ℝ + superscript ℝ \mathbb{R}^{+} .

As an example where the range equals the codomain, consider the function f ( x ) = 2 x f x 2 x f(x)=2x , which inputs a real number and outputs its double. For this function, the codomain and the image are the same (the function is a surjection), so the word range is unambiguous; it is the set of all real numbers.

Formal definition

When "range" is used to mean "codomain", the range of a function must be specified. It is often assumed to be the set of all real numbers, and {y | there exists an x in the domain of f such that y = f(x)} is called the image of f.

When "range" is used to mean "image", the range of a function f is {y | there exists an x in the domain of f such that y = f(x)}. In this case, the codomain of f must be specified, but is often assumed to be the set of all real numbers.

In both cases, image f ⊆ range f ⊆ codomain f, with at least one of the containments being equality.
