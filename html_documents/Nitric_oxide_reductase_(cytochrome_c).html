Nitric oxide reductase (cytochrome c)

Nitric oxide reductase (cytochrome c) () is an enzyme with system name nitrous oxide:ferricytochrome-c oxidoreductase.123456 This enzyme catalyses the following chemical reaction

nitrous oxide + 2 ferricytochrome c + H2O ⇌ normal-⇌ \rightleftharpoons 2 nitric oxide + 2 ferrocytochrome c + 2 H+

The enzyme from Pseudomonas aeruginosa contains a dinuclear centre.
