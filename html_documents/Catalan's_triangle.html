Catalan's triangle

In combinatorial mathematics, Catalan's triangle is a number triangle whose entries C ( n , k ) C n k C(n,k) give the number of strings consisting of n X's and k Y's such that no initial segment of the string has more Y's than X's. It is a generalization of the Catalan numbers, and is named after Eugène Charles Catalan.

Generalization

Catalan's trapezoids are a countable set of number trapezoids which generalize Catalan’s triangle. Catalan's trapezoid of order m=1,2,3,... is a number trapezoid whose entries C m ( n , k ) subscript C m n k C_{m}(n,k) give the number of strings consisting of n X's and k Y's such that in every initial segment of the string the number of Y's does not exceed the number of X's by m or more.3 By definition, Catalan's trapezoid of order m=1 is Catalan's triangle, i.e., C 1 ( n , k ) = C ( n , k ) subscript C 1 n k C n k C_{1}(n,k)=C(n,k) .
