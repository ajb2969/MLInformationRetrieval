Basic properties

  • The complement of a GŒ¥ set is an FœÉ set.
  • The intersection of countably many GŒ¥ sets is a GŒ¥ set, and the union of finitely many GŒ¥ sets is a GŒ¥ set; a countable union of GŒ¥ sets is called a GŒ¥œÉ set.
  • In metrizable spaces, every closed set is a GŒ¥ set and, dually, every open set is an FœÉ set.
  • A subspace A of a completely metrizable space X is itself completely metrizable if and only if A is a GŒ¥ set in X.
  • A set that contains the intersection of a countable collection of dense open sets is called comeagre or residual. These sets are used to define generic properties of topological spaces of functions.

GŒ¥ space

A GŒ¥ space is a topological space in which every closed set is a GŒ¥ set . A normal space which is also a GŒ¥ space is perfectly normal. Every metrizable space is perfectly normal, and every perfectly normal space is completely normal: neither implication is reversible.
