Law of total probability

In probability theory, the law (or formula) of total probability is a fundamental rule relating marginal probabilities to conditional probabilities. It expresses the total probability of an outcome which can be realized via several distinct events - hence the name.

Informal formulation

The above mathematical statement might be interpreted as follows: given an outcome A A A , with known conditional probabilities given any of the B n subscript B n B_{n} events, each with a known probability itself, what is the total probability that A A A will happen?. The answer to this question is given by Pr ( A ) Pr A \Pr(A) .

Example

Applications

One common application of the law is where the events coincide with a discrete random variable X taking each value in its range, i.e. B n subscript B n B_{n} is the event X = x n X subscript x n X=x_{n} . It follows that the probability of an event A is equal to the expected value of the conditional probabilities of A given X = x n X subscript x n X=x_{n} . That is,

Pr ( A ) = ∑ n Pr ( A ∣ X = x n ) Pr ( X = x n ) = E [ Pr ( A ∣ X ) ] , Pr A subscript n Pr A X subscript x n Pr X subscript x n normal-E Pr A X \Pr(A)=\sum_{n}\Pr(A\mid X=x_{n})\Pr(X=x_{n})=\operatorname{E}[\Pr(A\mid X)],

where Pr(A | X) is the conditional probability of A given the value of the random variable X.5 This conditional probability is a random variable in its own right, whose value depends on that of X. The conditional probability Pr(A | X = x) is simply a conditional probability given an event, [X = x]. It is a function of x, say g(x) = Pr(A | X = x). Then the conditional probability Pr(A | X) is g(X), hence itself a random variable. This version of the law of total probability says that the expected value of this random variable is the same as Pr(A).

This result can be generalized to continuous random variables (via continuous conditional density), and the expression becomes

Pr ( A ) = E [ Pr ( A ∣ ℱ X ) ] , Pr A normal-E Pr A subscript ℱ X \Pr(A)=\operatorname{E}[\Pr(A\mid\mathcal{F}_{X})],

where ℱ X subscript ℱ X \mathcal{F}_{X} denotes the sigma-algebra generated by the random variable X.

Other names

The term law of total probability is sometimes taken to mean the law of alternatives, which is a special case of the law of total probability applying to discrete random variables. One author even uses the terminology "continuous law of alternatives" in the continuous case.6 This result is given by Grimmett and Welsh7 as the partition theorem, a name that they also give to the related law of total expectation.
