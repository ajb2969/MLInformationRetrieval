Empirical distribution function

In statistics, the empirical distribution function is the distribution function associated with the empirical measure of the sample. This cdf is a step function that jumps up by 1/n at each of the n data points. The empirical distribution function estimates the underlying cdf of the points in the sample and converges with probability 1 according to the Glivenko‚ÄìCantelli theorem. However, it is important to note that the true point probability of a cdf jumps up by 1/(n+1).1

A number of results exist to quantify the rate of convergence of the empirical distribution function to the underlying cdf.
