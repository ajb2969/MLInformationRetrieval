Canonical cover

A canonical cover F c subscript F c F_{c} for F (a set of functional dependencies on a relation scheme) is a set of dependencies such that F logically implies all dependencies in F c subscript F c F_{c} , and F c subscript F c F_{c} logically implies all dependencies in F.

The set F c subscript F c F_{c} has two important properties:

  1. No functional dependency in F c subscript F c F_{c} contains an extraneous attribute.
  2. Each left side of a functional dependency in F c subscript F c F_{c} is unique. That is, there are no two dependencies a → b normal-→ a b a\to b and c → d normal-→ c d c\to d in F c subscript F c F_{c} such that a = c a c a=c .
