Relative accessible surface area

Relative accessible surface area or relative accessibility of a protein residue is a measure of residue solvent exposure. It can be calculated by formula:

R e l A c c = 100 * A c c / M a x A c c R e l A c c 100 A c c M a x A c c RelAcc=100*Acc/MaxAcc

  

where A c c A c c Acc is the solvent accessible surface area and M a x A c c M a x A c c MaxAcc is the maximum possible solvent accessible surface area for the residue in Å 2 superscript Å 2 {\AA}^{2} .
