Module homomorphism

In algebra, a module homomorphism is a function between modules that preserves module structures. Explicitly, if M and N are left modules over a ring R, then a function f : M → N normal-: f normal-→ M N f:M\to N is called a module homomorphism or a R-linear map if for any x, y in M and r in R,

f ( x + y ) = f ( x ) + f ( y ) , f x y f x f y f(x+y)=f(x)+f(y),

f ( r x ) = r f ( x ) . f r x r f x f(rx)=rf(x). If M, N are right module, then the second condition is replaced with

f ( x r ) = f ( x ) r . f x r f x r f(xr)=f(x)r.

The pre-image of the zero element under f is called the kernel of f. The set of all module homomorphisms from M to N is denoted by HomR(M, N). It is an abelian group but is not necessarily a module unless R is commutative.

The isomorphism theorems hold for module homomorphisms.
