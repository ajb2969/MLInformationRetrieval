Knödel number

In number theory, a Knödel number for a given positive integer n is a composite number m with the property that each i i^{m - n} \equiv 1 \pmod{m}. The concept is named after Walter Knödel.1 The set of all Knödel numbers for n is denoted Kn.

The special case K1 are the Carmichael numbers.

Every composite number is a Knödel number. This can be seen by setting n n n equal to m - φ ( n ) m φ n m-\varphi(n) .
