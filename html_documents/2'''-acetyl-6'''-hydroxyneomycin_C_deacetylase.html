2'''-acetyl-6'''-hydroxyneomycin C deacetylase (, neoL (gene)) is an enzyme with system name 2'''-acetyl-6'''-hydroxyneomycin C hydrolase (acetate-forming).1 This enzyme catalyses the following chemical reaction

2'''-acetyl-6'''-deamino-6'''-hydroxyneomycin C + H2O $\rightleftharpoons$ 6'''-deamino-6'''-hydroxyneomycin C + acetate

This enzyme is involved in biosynthesys of aminoglycoside antibiotics of the [neomycin] family.
