Alternative usage

Some authors use the phrase "indefinite sum" to describe a sum in which the numerical value of the upper limit is not given. e.g.

∑ k = 1 n f ( k ) superscript subscript k 1 n f k \sum_{k=1}^{n}f(k)

In this case a closed form expression F(k) for the sum is a solution of

F ( x + 1 ) - F ( x ) = f ( x + 1 ) F x 1 F x f x 1 F(x+1)-F(x)=f(x+1)\, which is called the telescoping equation.9 It is inverse to backward difference ∇ normal-∇ \nabla operator. It is related to the forward antidifference operator using the fundamental theorem of discrete calculus described earlier.
