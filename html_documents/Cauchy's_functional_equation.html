Cauchy's functional equation

Cauchy's functional equation is the functional equation

f ( x + y ) = f ( x ) + f ( y ) . f x y f x f y f(x+y)=f(x)+f(y).

Solutions to this are called additive functions. Over the rational numbers, it can be shown using elementary algebra that there is a single family of solutions, namely f ( x ) = c x f x c x f(x)=cx for any arbitrary rational number c c c . Over the real numbers, this is still a family of solutions; however there can exist other solutions that are extremely complicated. Further constraints on f sometimes preclude other solutions, for example:

  • if f f f is continuous (proven by Cauchy in 1821). This condition was weakened in 1875 by Darboux who showed that it was only necessary for the function to be continuous at one point.
  • if f f f is monotonic on any interval.
  • if f f f is bounded on any interval.

On the other hand, if no further conditions are imposed on f f f , then (assuming the axiom of choice) there are infinitely many other functions that satisfy the equation. This was proved in 1905 by Georg Hamel using Hamel bases. Such functions are sometimes called Hamel functions.1

The fifth problem on Hilbert's list is a generalisation of this equation. Functions where there exists a real number c c c such that f ( c x ) ≠ c f ( x ) f c x c f x f(cx)\neq cf(x) are known as Cauchy-Hamel functions and are used in Dehn-Hadwiger invariants which are used in the extension of Hilbert's third problem from 3-D to higher dimensions.2
