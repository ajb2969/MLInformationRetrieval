Bid–ask matrix

The bid–ask matrix is a matrix with elements corresponding with exchange rates between the assets. These rates are in physical units (e.g. number of stocks) and not with respect to any numeraire. The ( i , j ) i j (i,j) element of the matrix is the number of units of asset i i i which can be exchanged for 1 unit of asset j j j .
