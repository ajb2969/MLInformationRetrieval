Empirical evidence

Imran Ghory has used data on the largest prime gaps to confirm the conjecture for n n n up to 1.3002 x 1016.2 Using a table of maximal gaps and the above gap inequality, the confirmation value can be extended exhaustively to 4 x 1018.

Since the Andrica function decreases asymptotically as n increases, a prime gap of ever increasing size is needed to make the difference large as n becomes large. It therefore seems highly likely the conjecture is true, although this has not yet been proven.
