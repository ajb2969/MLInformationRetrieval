Fxgrep

fxgrep (The Functional XML Querying Tool) is an XML querying tool written in the Standard ML programming language. The Fxgrep addresses the problem of dealing with k k k -ary queries, that is, the task of simultaneously locating k k k nodes of an input XML tree that satisfy a given relation.1
