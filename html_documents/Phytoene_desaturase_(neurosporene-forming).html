Phytoene desaturase (neurosporene-forming)

Phytoene desaturase (neurosporene-forming) (, 3-step phytoene desaturase, three-step phytoene desaturase, phytoene desaturase (ambiguous), CrtI (ambiguous)) is an enzyme with system name 15-cis-phytoene:acceptor oxidoreductase (neurosporene-forming).12 This enzyme catalyses the following chemical reaction

15-cis-phytoene + 3 acceptor ⇌ normal-⇌ \rightleftharpoons all-trans-neurosporene + 3 reduced acceptor (overall reaction)
(1a) 15-cis-phytoene + acceptor ⇌ normal-⇌ \rightleftharpoons all-trans-phytofluene + reduced acceptor
(1b) all-trans-phytofluene + acceptor ⇌ normal-⇌ \rightleftharpoons all-trans-zeta-carotene + reduced acceptor
(1c) all-trans-zeta-carotene + acceptor ⇌ normal-⇌ \rightleftharpoons all-trans-neurosporene + reduced acceptor

This enzyme is involved in carotenoid biosynthesis.
