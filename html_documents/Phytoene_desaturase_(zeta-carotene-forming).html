Phytoene desaturase (zeta-carotene-forming)

Phytoene desaturase (zeta-carotene-forming) (, CrtIa, 2-step phytoene desaturase (ambiguous), two-step phytoene desaturase (ambiguous)) is an enzyme with system name 15-cis-phytoene:acceptor oxidoreductase (zeta-carotene-forming).1 This enzyme catalyses the following chemical reaction

15-cis-phytoene + 2 acceptor ⇌ normal-⇌ \rightleftharpoons all-trans-zeta-carotene + 2 reduced acceptor (overall reaction)
(1a) 15-cis-phytoene + acceptor ⇌ normal-⇌ \rightleftharpoons all-trans-phytofluene + reduced acceptor
(1b) all-trans-phytofluene + acceptor ⇌ normal-⇌ \rightleftharpoons all-trans-zeta-carotene + reduced acceptor

The enzyme is involved in carotenoid biosynthesis.
