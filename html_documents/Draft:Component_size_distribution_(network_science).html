Component size distribution (network science)

Component size distribution is the basic property of undirected graph. The property focuses on the sizes of connected components of vertices in the graph in the regime without a giant component.2 Distribution of component size shows the number of smaller components expected for a given number of edges, ⟨ k ⟩ delimited-⟨⟩ k {\left\langle k\right\rangle} , and their expected sizes.

In a peer to peer network

Percolation threshold

The stability of networks are primarily measured in terms of certain fraction of nodes ( f c ) subscript f c {(f_{c})} called percolation threshold. Its removal disintegrates the network into large number of small, disconnected components.10 Above that threshold, there exists a connected component which goes through the entire network. This connected component is also termed as the giant component. The value of percolation threshold f c subscript f c {f_{c}} theoretically signifies the stability of the network, higher value indicates greater stability against churn and attack.11
It is possible to develop the metric to measure the percolation threshold.12 To do this, a fraction of nodes f t subscript f t {f_{t}} should be removed from the network. Then, after each step t t {t} one should check whether the percolation network is reached.
