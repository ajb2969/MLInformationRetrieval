Formate dehydrogenase-N

Formate dehydrogenase-N (, Fdh-N, FdnGHI, nitrate-inducible formate dehydrogenase, formate dehydrogenase N, FDH-N, nitrate inducible Fdn, nitrate inducible formate dehydrogenase) is an enzyme with system name formate:quinone oxidoreductase.123 This enzyme catalyses the following chemical reaction

formate + quinone ⇌ normal-⇌ \rightleftharpoons CO2 + quinol

This enzyme contains molybdopterin-guanine dinucleotides, five [4Fe-4S] clusters and two heme b groups.
