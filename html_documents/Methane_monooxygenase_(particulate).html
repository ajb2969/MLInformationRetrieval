Methane monooxygenase (particulate)

Methane monooxygenase (particulate) () is an enzyme with system name methane,quinol:oxygen oxidoreductase.1234 This enzyme catalyses the following chemical reaction

methane + quinol + O2 ⇌ normal-⇌ \rightleftharpoons methanol + quinone + H2O

Methane monooxygenase contains copper. It is membrane-bound.
