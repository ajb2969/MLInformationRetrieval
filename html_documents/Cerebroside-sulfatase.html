Cerebroside-sulfatase

Cerebroside-sulfatase (, arylsulfatase A, cerebroside sulfate sulfatase) is an enzyme with system name cerebroside-3-sulfate 3-sulfohydrolase.12 This enzyme catalyses the following chemical reaction

a cerebroside 3-sulfate + H2O ⇌ normal-⇌ \rightleftharpoons a cerebroside + sulfate

This enzyme hydrolyses galactose-3-sulfate residues in a number of lipids.
