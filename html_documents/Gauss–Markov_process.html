Gauss‚ÄìMarkov stochastic processes (named after Carl Friedrich Gauss and Andrey Markov) are stochastic processes that satisfy the requirements for both Gaussian processes and Markov processes.12 The stationary Gauss‚ÄìMarkov process is a very special case because it is unique, except for some trivial exceptions.

Every Gauss‚ÄìMarkov process X(t) possesses the three following properties:

  1. If h(t) is a non-zero scalar function of t, then Z(t) = h(t)X(t) is also a Gauss‚ÄìMarkov process
  2. If f(t) is a non-decreasing scalar function of t, then Z(t) = X(f(t)) is also a Gauss‚ÄìMarkov process
  3. There exists a non-zero scalar function h(t) and a non-decreasing scalar function f(t) such that X(t) = h(t)W(f(t)), where W(t) is the standard Wiener process.

Property (3) means that every Gauss‚ÄìMarkov process can be synthesized from the standard Wiener process (SWP).
