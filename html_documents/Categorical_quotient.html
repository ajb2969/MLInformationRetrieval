Categorical quotient

In algebraic geometry, given a category C, a categorical quotient of an object X with action of a group G is a morphism π : X → Y normal-: π normal-→ X Y \pi:X\to Y that

(i) is invariant; i.e., π ∘ σ = π ∘ p 2 π σ π subscript p 2 \pi\circ\sigma=\pi\circ p_{2} where σ : G × X → X normal-: σ normal-→ G X X \sigma:G\times X\to X is the given group action and p2 is the projection.
(ii) satisfies the universal property: any morphism X → Z normal-→ X Z X\to Z satisfying (i) uniquely factors through π π \pi .

One of the main motivations for the development of geometric invariant theory was the construction of a categorical quotient for varieties or schemes.

Note π π \pi need not be surjective. Also, if it exists, a categorical quotient is unique up to a canonical isomorphism. In practice, one takes C to be the category of varieties or the category of schemes over a fixed scheme. A categorical quotient π π \pi is a universal categorical quotient if it is stable under base change: for any Y ′ → Y normal-→ superscript Y normal-′ Y Y^{\prime}\to Y , π ′ : X ′ = X × Y Y ′ → Y ′ normal-: superscript π normal-′ superscript X normal-′ subscript Y X superscript Y normal-′ normal-→ superscript Y normal-′ \pi^{\prime}:X^{\prime}=X\times_{Y}Y^{\prime}\to Y^{\prime} is a categorical quotient.

A basic result is that geometric quotients (e.g., G / H G H G/H ) and GIT quotients (e.g., X / / G fragments X G X/\!/G ) are categorical quotients.
