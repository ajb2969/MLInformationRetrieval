Finite unions

One can take the union of several sets simultaneously. For example, the union of three sets A, B, and C contains all elements of A, all elements of B, and all elements of C, and nothing else. Thus, x is an element of A âˆª B âˆª C if and only if x is in at least one of A, B, and C.

In mathematics a finite union means any union carried out on a finite number of sets: it doesn't imply that the union set is a finite set.
