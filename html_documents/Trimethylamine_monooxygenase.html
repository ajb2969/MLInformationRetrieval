Trimethylamine monooxygenase

Trimethylamine monooxygenase (, flavin-containing monooxygenase 3, FMO3, tmm (gene)) is an enzyme with system name N,N,N-trimethylamine,NADPH:oxygen oxidoreductase (N-oxide-forming).1234 This enzyme catalyses the following chemical reaction

N,N,N-trimethylamine + NADPH + H+ + O2 ⇌ normal-⇌ \rightleftharpoons N,N,N-trimethylamine N-oxide + NADP+ + H2O

Trimethylamine monooxygenase is a flavoprotein.
