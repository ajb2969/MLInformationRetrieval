Usage

2 2 \sqrt{2} gives 2 2 \sqrt{2} .

This template is an alternative to using the LaTeX \sqrt rendered in-line, and will make the square root appear as text with the √ symbol and then the number (entered as the first parameter in this template) with an overline. Do not use this inside tags.
