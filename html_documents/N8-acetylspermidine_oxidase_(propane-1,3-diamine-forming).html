N8-acetylspermidine oxidase (propane-1,3-diamine-forming)

N8-acetylspermidine oxidase (propane-1,3-diamine-forming) () is an enzyme with system name N8-acetylspermidine:oxygen oxidoreductase (propane-1,3-diamine-forming).1 This enzyme catalyses the following chemical reaction

N8-acetylspermidine + O2 + H2O ⇌ normal-⇌ \rightleftharpoons propane-1,3-diamine + 4-acetamidobutanal + H2O2

This enzyme is also active n N1-acetylspermine, and it has weak activity on N1,N12-diacetylspermine.
