Square-lattice Ising model

In statistical mechanics, the two-dimensional square-lattice Ising model is a simple model of interacting magnetic spins. The model is notable for having nontrivial interactions, yet having an analytical solution. The model was solved by for the special case that the external magnetic field H = 0. An analytical solution for the general case for H ≠ 0 H 0 H\neq 0 has yet to be found.
