Disk covering problem

The disk covering problem asks for the smallest real number r ( n ) r n r(n) such that n n n disks of radius r ( n ) r n r(n) can be arranged in such a way as to cover the unit disk. Dually, for a given radius ε, one wishes to find the smallest integer n such that n disks of radius ε can cover the unit disk.1
