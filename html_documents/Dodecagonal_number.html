Dodecagonal number

A dodecagonal number is a figurate number that represents a dodecagon. The dodecagonal number for n is given by the formula

The dodecagonal number for n can also be calculated by adding the square of n to four times the (n - 1)th pronic number, or to put it algebraically, D n = n 2 + 4 ( n 2 - n ) subscript D n superscript n 2 4 superscript n 2 n D_{n}=n^{2}+4(n^{2}-n) .

Dodecagonal numbers consistently alternate parity, and in base 10, their units place digits follow the pattern 1, 2, 3, 4, 5, 6, 7, 8, 9, 0.

By the Fermat polygonal number theorem, every number is the sum of at most 12 dodecagonal numbers.

Dodecagonal Square numbers are numbers that are both a perfect square and a dodecagonal number. There are an infinite number of dodecagonal square numbers. The first few are:
