(+)-Sabinene 3-hydroxylase

(+)-Sabinene 3-hydroxylase () is an enzyme with system name (+)-sabinene,NADPH:oxygen oxidoreductase (3-hydroxylating).1 This enzyme catalyses the following chemical reaction

(+)-sabinene + NADPH + H+ + O2 ⇌ normal-⇌ \rightleftharpoons (+)-cis-sabinol + NADP+ + H2O

(+)-Sabinene 3-hydroxylase requires cytochrome P-450.
