Vaughan's identity

In mathematics, Vaughan's identity is an identity found by that can be used to simplify Vinogradov's work on trigonometric sums. It can be used to estimate sums of the form

∑ f ( n ) Λ ( n ) f n normal-Λ n \sum f(n)\Lambda(n) where f is some function of positive integers n, whose values in applications are often roots of unity, and Λ is the von Mangoldt function.

Vaughan's identity has been used to simplify the proof of the Bombieri–Vinogradov theorem and to study Kummer sums.
