An open conjecture

It is known, that the harmonic numbers are never integers except the case n=1. The same question can be posed with respect to the hyperharmonic numbers: are there integer hyperharmonic numbers? István Mező proved6 that if r=2 or r=3, these numbers are never integers except the trivial case when n=1. He conjectured that this is always the case, namely, the hyperharmonic numbers of order r are never integers except when n=1. This conjecture was justified for a class of parameters by R.
