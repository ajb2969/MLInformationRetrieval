Dinatural transformation

In category theory, a dinatural transformation α α \alpha between two functors

S , T : C op × C → X , normal-: S T normal-→ superscript C op C X S,T:C^{\mathrm{op}}\times C\to X,

written

α : S → ¨ T , normal-: α S normal-¨ normal-→ T \alpha:S\ddot{\to}T,

is a function which to every object c of C associates an arrow

α c : S ( c , c ) → T ( c , c ) normal-: subscript α c normal-→ S c c T c c \alpha_{c}:S(c,c)\to T(c,c) of X

and satisfies the following coherence property: for every morphism f : c → c ′ normal-: f normal-→ c superscript c normal-′ f:c\to c^{\prime} of C the diagram commutes.

The composition of two dinatural transformations need not be dinatural.
