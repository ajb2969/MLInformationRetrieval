Configuration state function

In quantum chemistry, a configuration state function (CSF), is a symmetry-adapted linear combination of Slater determinants. A CSF must not be confused with a configuration. In general, one configuration gives rise to several CSFs; all have the same total quantum numbers for spin and spatial parts but differ in their intermediate couplings.

Definition

In quantum chemistry, a configuration state function (CSF), is a symmetry-adapted linear combination of Slater determinants. It is constructed to have the same quantum numbers as the wavefunction, Ψ normal-Ψ \Psi , of the system being studied. In the method of configuration interaction the wavefunction1 can be expressed as a linear combination of CSFs, that is in the form

Ψ = ∑ k c k ψ k normal-Ψ subscript k subscript c k subscript ψ k \Psi=\sum_{k}c_{k}\psi_{k}

where ψ k subscript ψ k \psi_{k} denotes the set of CSFs. The coefficients, c k subscript c k c_{k} , are found by using the expansion of Ψ normal-Ψ \Psi to compute a Hamiltonian matrix. When this is diagonalized, the eigenvectors are chosen as the expansion coefficients. CSFs rather than just Slater determinants can also be used as a basis in Multi-configurational self-consistent field computations.

In atomic structure, a CSF is an eigenstate of

  • the square of the angular momentum operator, L ^ 2 superscript normal-^ L 2 \hat{L}^{2} .
  • the z-projection of angular momentum L ^ z subscript normal-^ L z \hat{L}_{z}
  • the square of the spin operator S ^ 2 superscript normal-^ S 2 \hat{S}^{2} .
  • the z-projection of the spin operator S ^ z subscript normal-^ S z \hat{S}_{z}

In linear molecules, L ^ 2 superscript normal-^ L 2 \hat{L}^{2} does not commute with the Hamiltonian for the system and therefore CSFs are not eigenstates of L ^ 2 superscript normal-^ L 2 \hat{L}^{2} . However, the z-projection of angular momentum is still a good quantum number and CSFs are constructed to be eigenstates of L ^ z , S ^ 2 subscript normal-^ L z superscript normal-^ S 2 \hat{L}_{z},\hat{S}^{2} and S ^ z subscript normal-^ S z \hat{S}_{z} . In non-linear (which implies polyatomic) molecules, neither L ^ 2 superscript normal-^ L 2 \hat{L}^{2} nor L ^ z subscript normal-^ L z \hat{L}_{z} commute with the Hamiltonian. The CSFs are constructed to have the spatial transformation properties of one of the irreducible representations of the point group to which the nuclear framework belongs. This is because the Hamiltonian operator transforms in the same way.2 S ^ 2 superscript normal-^ S 2 \hat{S}^{2} and S ^ z subscript normal-^ S z \hat{S}_{z} are still valid quantum numbers and CSFs are built to be eigenfunctions of these operators.

From Configurations to Configuration State Functions

CSFs are however derived from configurations. A configuration is just an assignment of electrons to orbitals. For example 1 s 2 1 superscript s 2 1s^{2} and 1 π 2 1 superscript π 2 1\pi^{2} are example of two configurations, one from atomic structure and one from molecular structure.

From any given configuration we can, in general, create several CSFs. CSFs are therefore sometimes also called N-particle symmetry adapted basis functions. It is important to realize that for a configuration the number of electrons is fixed; let's call this N N N . When we are creating CSFs from a configuration we have to work with the spin-orbitals associated with the configuration.

For example given the 1 s 1 s 1s orbital in an atom we know that there are two spin-orbitals associated with this,

1 s α 1 s β 1 s α 1 s β 1s\alpha\;\;\;1s\beta

where

α , β α β \alpha,\;\;\;\beta

are the one electron spin-eigenfunctions for spin-up and spin-down respectively. Similarly, for the 1 π 1 π 1\pi orbital in a linear molecule ( C ∞ v subscript C v C_{\infty v} point group) we have four spin orbitals:

1 π ( + ) α , 1 π ( + ) β , 1 π ( - ) α , 1 π ( - ) β 1 π α 1 π β 1 π α 1 π β 1\pi(+)\alpha,\;1\pi(+)\beta,\;1\pi(-)\alpha,\;1\pi(-)\beta .

This is because the π π \pi designation corresponds to z-projection of angular momentum of both + 1 1 +1 and - 1 1 -1 .

We can think of the set of spin orbitals as a set of boxes each of size one; let's call this M M M boxes. We distribute the N N N electrons among the M M M boxes in all possible ways. Each assignment corresponds to one Slater determinant, D i subscript D i D_{i} . There can be great number of these, particularly when N ≪ M much-less-than N M N< . Another way to look at this is to say we have M M M entities and we wish to select N N N of them, known as a combination. We need to find all possible combinations. Order of the selection is not significant because we are working with determinants and can interchange rows as required.

If we then specify the overall coupling that we wish to achieve for the configuration, we can now select only those Slater determinants that have the required quantum numbers. In order to achieve the required total spin angular momentum (and in the case of atoms the total orbital angular momentum as well), each Slater determinant has to be premultiplied by a coupling coefficient c i subscript c i c_{i} , derived ultimately from Clebsch–Gordan coefficients. Thus the CSF is a linear combination

∑ i c i D i subscript i subscript c i subscript D i \sum_{i}c_{i}\;D_{i} .

The Lowdin projection operator formalism3 may be used to find the coefficients. For any given set of determinants D i subscript D i D_{i} it may be possible to find several different sets of coefficients.4 Each set corresponds to one CSF. In fact this simply reflects the different internal couplings of total spin and spatial angular momentum.

A genealogical algorithm for CSF construction

At the most fundamental level, a configuration state function can be constructed

  • from a set of M M M orbitals

and

  • a number N N N of electrons

using the following genealogical algorithm:

  1. distribute the N N N electrons over the set of M M M orbitals giving a configuration
  2. for each orbital the possible quantum number couplings (and therefore wavefunctions for the individual orbitals) are known from basic quantum mechanics; for each orbital choose one of the permitted couplings but leave the z-component of the total spin, S z subscript S z S_{z} undefined.
  3. check that the spatial coupling of all orbitals matches that required for the system wavefunction. For a molecule exhibiting C ∞ v subscript C v C_{\infty v} or D ∞ h subscript D h D_{\infty h} this is achieved by a simple linear summation of the coupled λ λ \lambda value for each orbital; for molecules whose nuclear framework transforms according to D 2 h subscript D 2 h D_{2h} symmetry, or one of its sub-groups, the group product table has to be used to find the product of the irreducible representation of all N N N orbitals.
  4. couple the total spins of the N N N orbitals from left to right; this means we have to choose a fixed S z subscript S z S_{z} for each orbital.
  5. test the final total spin and its z-projection against the values required for the system wavefunction

The above steps will need to be repeated many times to elucidate the total set of CSFs that can be derived from the N N N electrons and M M M orbitals.

Single Orbital configurations and wavefunctions

Basic quantum mechanics defines the possible single orbital wavefunctions. In a software implementation, these can be provided either as a table or through a set of logic statements. Alternatively group theory may be used to compute them 5 Electrons in a single orbital are called equivalent electrons. 6 They obey the same coupling rules as other electrons but the Pauli exclusion principle makes certain couplings impossible. The Pauli exclusion principle requires that no two electrons in a system can have all their quantum numbers equal. For equivalent electrons, by definition the principal quantum number is identical. In atoms the angular momentum is also identical. So, for equivalent electrons the z components of spin and spatial parts, taken together, must differ.
