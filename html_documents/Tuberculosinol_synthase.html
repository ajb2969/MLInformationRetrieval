Tuberculosinol synthase

Tuberculosinol synthase (, Rv3378c) is an enzyme with system name tuberculosinyl diphosphate diphosphohydrolase (tuberculosinol forming).12 This enzyme catalyses the following chemical reaction

tuberculosinyl diphosphate + H2O ⇌ normal-⇌ \rightleftharpoons [tuberculosinol] + diphosphate

This enzyme is present in Mycobacterium that cause tuberculosis.
