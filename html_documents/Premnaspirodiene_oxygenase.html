Premnaspirodiene oxygenase

Premnaspirodiene oxygenase (, HPO, Hyoscymus muticus premnaspirodiene oxygenase) is an enzyme with system name (-)-vetispiradiene,NADPH:oxygen 2alpha-oxidoreductase.1 This enzyme catalyses the following chemical reaction

(-)-vetispiradiene + 2 NADPH + 2 H+ + 2 O2 ⇌ normal-⇌ \rightleftharpoons solavetivone + 2 NADP+ + 3 H2O (overall reaction)
(1a) (-)-vetispiradiene + NADPH + H+ + O2 ⇌ normal-⇌ \rightleftharpoons solavetivol + NADP+ + H2O
(1b) solavetivol + NADPH + H+ + O2 ⇌ normal-⇌ \rightleftharpoons solavetivone + NADP+ + 2 H2O

Premnaspirodiene oxygenase heme-thiolate protein (P-450).
