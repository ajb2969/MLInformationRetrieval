Applications

Perifocal reference frames are most commonly used with elliptical orbits for the reason that the 𝕡 ^ normal-^ 𝕡 \mathbb{\hat{p}} coordinate must be aligned with the eccentricity vector. Circular Orbits, having no eccentricity, give no means by which to orient the coordinate system about the focus.5

The perifocal coordinate system may also be uses as an inertial frame of reference because the axes do not rotate relative to the fixed stars. This allows the inertia of any orbital bodies within this frame of reference to be calculated. This is useful when attempting to solve problems like the two-body problem.6
