Glycine oxidase

Glycine oxidase () is an enzyme with system name glycine:oxygen oxidoreductase (deaminating).12 This enzyme catalyses the following chemical reaction

glycine + H2O + O2 ⇌ normal-⇌ \rightleftharpoons glyoxylate + NH3 + H2O2 (overall reaction)
(1a) glycine + O2 ⇌ normal-⇌ \rightleftharpoons 2-iminoacetate + H2O2
(1b) 2-iminoacetate + H2O ⇌ normal-⇌ \rightleftharpoons glyoxylate + NH3

This flavoenzyme containing non-covalently bound FAD.
