Ball (mathematics)

In mathematics, a ball is the space inside a sphere. It may be a closed ball (including the boundary points of the sphere) or an open ball (excluding them).

These concepts are defined not only in three-dimensional Euclidean space but also for lower and higher dimensions, and for metric spaces in general. A ball in n n n dimensions is called an n n n -ball and is bounded by an ( n n n -1)-sphere. Thus, for example, a ball in the Euclidean plane is the same thing as a disk, the area bounded by a circle. In Euclidean 3-space, a ball is taken to be the volume bounded by a 2-dimensional spherical shell boundary.

In other contexts, such as in Euclidean geometry and informal use, sphere is sometimes used to mean ball.

Balls in Euclidean space

In Euclidean n n n -space, an (open) n n n -ball of radius r r r and center x x x is the set of all points of distance $r$ from x x x . A closed n n n -ball of radius r r r is the set of all points of distance ≤  r r r away from x x x .

In Euclidean n n n -space, every ball is the interior of a hypersphere (a hyperball), that is a bounded interval when n n n  = 1, the interior of a circle (a disk) when n n n  = 2, and the interior of a sphere when n n n  = 3.

A (open or closed) unit ball is a ball of radius 1.

A subset of a metric space is bounded if it is contained in some ball. A set is totally bounded if, given any positive radius, it is covered by finitely many balls of that radius.

The open balls of a metric space are a basis for a topological space, whose open sets are all possible unions of open balls. This space is called the topology induced by the metric d d d .

Topological balls

One may talk about balls in any topological space X X X , not necessarily induced by a metric. An (open or closed) n n n -dimensional topological ball of X X X is any subset of X X X which is homeomorphic to an (open or closed) Euclidean n n n -ball. Topological n n n -balls are important in combinatorial topology, as the building blocks of cell complexes.

Any open topological n n n -ball is homeomorphic to the Cartesian space R n n n and to the open unit n n n -cube (hypercube) ( 0 , 1 ) n ⊆ \R n superscript 0 1 n superscript \R n (0,1)^{n}\subseteq\R^{n} . Any closed topological n n n -ball is homeomorphic to the closed n n n -cube [0, 1] n n n .

An n n n -ball is homeomorphic to an m m m -ball if and only if n n n  =  m m m . The homeomorphisms between an open n n n -ball B B B and R n n n can be classified in two classes, that can be identified with the two possible topological orientations of  B B B .

A topological n n n -ball need not be smooth; if it is smooth, it need not be diffeomorphic to a Euclidean n n n -ball.
