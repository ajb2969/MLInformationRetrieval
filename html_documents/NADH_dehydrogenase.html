NADH dehydrogenase

NADH dehydrogenase (, cytochrome c reductase, type 1 dehydrogenase, beta-NADH dehydrogenase dinucleotide, diaphorase, dihydrocodehydrogenase I dehydrogenase, dihydronicotinamide adenine dinucleotide dehydrogenase, diphosphopyridine diaphorase, DPNH diaphorase, NADH diaphorase, NADH hydrogenase, NADH oxidoreductase, NADH-menadione oxidoreductase, reduced diphosphopyridine nucleotide diaphorase) is an enzyme with systematic name NADH:acceptor oxidoreductase.1234 This enzyme catalyses the following chemical reaction

NADH + H+ + acceptor ⇌ normal-⇌ \rightleftharpoons NAD+ + reduced acceptor

NADH dehydrogenase is a flavoprotein that contains iron-sulfur centres.
