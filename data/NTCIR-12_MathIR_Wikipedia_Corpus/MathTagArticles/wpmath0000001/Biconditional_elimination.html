Biconditional elimination

Biconditional elimination is the name of two valid rules of inference of propositional logic. It allows for one to infer a conditional from a biconditional. If ( P ↔ Q ) normal-↔ P Q (P\leftrightarrow Q) is true, then one may infer that ( P → Q ) normal-→ P Q (P\to Q) is true, and also that ( Q → P ) normal-→ Q P (Q\to P) is true.1 For example, if it's true that I'm breathing if and only if I'm alive, then it's true that if I'm breathing, I'm alive; likewise, it's true that if I'm alive, I'm breathing. The rules can be stated formally as:

( P ↔ Q ) ∴ ( P → Q ) normal-↔ P Q fragments therefore fragments normal-( P normal-→ Q normal-) \frac{(P\leftrightarrow Q)}{\therefore(P\to Q)} and

( P ↔ Q ) ∴ ( Q → P ) normal-↔ P Q fragments therefore fragments normal-( Q normal-→ P normal-) \frac{(P\leftrightarrow Q)}{\therefore(Q\to P)}

where the rule is that wherever an instance of " ( P ↔ Q ) normal-↔ P Q (P\leftrightarrow Q) " appears on a line of a proof, either " ( P → Q ) normal-→ P Q (P\to Q) " or " ( Q → P ) normal-→ Q P (Q\to P) " can be placed on a subsequent line;

Formal notation

The biconditional elimination rule may be written in sequent notation:

( P ↔ Q ) ⊢ ( P → Q ) fragments fragments normal-( P normal-↔ Q normal-) proves fragments normal-( P normal-→ Q normal-) (P\leftrightarrow Q)\vdash(P\to Q) and

( P ↔ Q ) ⊢ ( Q → P ) fragments fragments normal-( P normal-↔ Q normal-) proves fragments normal-( Q normal-→ P normal-) (P\leftrightarrow Q)\vdash(Q\to P)

where ⊢ proves \vdash is a metalogical symbol meaning that ( P → Q ) normal-→ P Q (P\to Q) , in the first case, and ( Q → P ) normal-→ Q P (Q\to P) in the other are syntactic consequences of ( P ↔ Q ) normal-↔ P Q (P\leftrightarrow Q) in some logical system;

or as the statement of a truth-functional tautology or theorem of propositional logic:

( P ↔ Q ) → ( P → Q ) fragments fragments normal-( P normal-↔ Q normal-) normal-→ fragments normal-( P normal-→ Q normal-) (P\leftrightarrow Q)\to(P\to Q)

( P ↔ Q ) → ( Q → P ) fragments fragments normal-( P normal-↔ Q normal-) normal-→ fragments normal-( Q normal-→ P normal-) (P\leftrightarrow Q)\to(Q\to P)

where P P P , and Q Q Q are propositions expressed in some formal system.
