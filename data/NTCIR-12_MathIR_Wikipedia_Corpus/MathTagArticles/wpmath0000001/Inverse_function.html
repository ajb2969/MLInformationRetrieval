Inverse function

In mathematics, an inverse function is a function that "reverses" another function. That is, if f f f is a function mapping x x x to y y y , then the inverse function of f f f maps y y y back to x x x .1

Definitions

Let f f f be a function whose domain is the set X X X , and whose image (range) is the set Y Y Y . Then f f f is invertible if there exists a function g g g with domain Y Y Y and image X X X , with the property:

f ( x ) = y ⇔ g ( y ) = x . normal-⇔ f x y g y x f(x)=y\,\,\Leftrightarrow\,\,g(y)=x.

If f f f is invertible, the function g g g is unique, which means that there is exactly one function g g g satisfying this property (no more, no less). That function g g g is then called the inverse of f f f , and is usually denoted as .

Stated otherwise, a function is invertible if and only if its inverse relation is a function on the range Y Y Y , in which case the inverse relation is the inverse function.2

Not all functions have an inverse. For this rule to be applicable, each element y ∈ Y y normal-∈ Y y∈Y must correspond to no more than one x ∈ X x normal-∈ X x∈X ; a function f f f with this property is called one-to-one or an injection. If f f f and are functions on X X X and Y Y Y respectively, then both are bijections. The inverse of an injection that is not a bijection is a partial function, that means for some y ∈ Y y normal-∈ Y y∈Y it is undefined.

Example: squaring and square root functions

The function may or may not be invertible, depending on what kinds of numbers are being considered (the "domain").

If the domain is the real numbers, then each possible result y (except 0) corresponds to two different starting points in X X X – one positive and one negative, and so this function is not invertible: as it is impossible to deduce an input from its output. Such a function is called non-injective or information-losing.

If the domain of the function is restricted to the nonnegative reals then the function is injective and invertible.

Inverses in higher mathematics

The definition given above is commonly adopted in set theory and calculus. In higher mathematics, the notation

f : X → Y normal-: f normal-→ X Y f\colon X\to Y means " f f f is a function mapping elements of a set X X X to elements of a set Y Y Y ". The source, X X X , is called the domain of f f f , and the target, Y Y Y , is called the codomain. The codomain contains the range of f f f as a subset, and is part of the definition of f f f .3

When using codomains, the inverse of a function f : X → Y normal-: f X normal-→ Y f:X→Y is required to have domain Y Y Y and codomain X X X . For the inverse to be defined on all of Y Y Y , every element of Y Y Y must lie in the range of the function f f f . A function with this property is called onto or surjective. Thus, a function with a codomain is invertible if and only if it is both injective (one-to-one) and surjective (onto). Such a function is called a one-to-one correspondence or a bijection, and has the property that every element y ∈ Y y normal-∈ Y y∈Y corresponds to exactly one element x ∈ X x normal-∈ X x∈X .

Uniqueness

If an inverse function exists for a given function f f f , it is unique: it must be the inverse relation.
